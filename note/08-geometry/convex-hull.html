凸包概述

凸包（Convex Hull）是包含点集中所有点的最小凸多边形。可以类比为：在平面上钉了 n 个钉子，用一根橡皮筋围住所有钉子，松手后橡皮筋形成的多边形就是凸包。

  凸包示意图

        * E
       / \
      /   \
   * D     * F          外层的 A-B-C-D-E-F-G 构成凸包
    |       |            内部的 * 不在凸包上
   * C     * G
    |       |
    |   * * |
     \     /
      * B * H
       \ /
        * A

  凸包上的点称为 hull vertices
  凸包内部的点不会出现在结果中

基本性质

  • 凸包是一个凸多边形：任意内角不超过 180°
  • 凸包是包含所有点的面积最小的凸多边形
  • 凸包是包含所有点的周长最小的凸多边形
  • 点集的凸包唯一（退化情况：所有点共线时退化为线段）

常见算法

算法时间复杂度特点
Andrew (单调链)O(n log n)实现简单，推荐首选
Graham ScanO(n log n)经典算法，需要极角排序
Gift WrappingO(nh)简单但慢，h 为凸包点数
Divide and ConquerO(n log n)分治思想
叉积在凸包中的作用

凸包算法的核心是利用叉积判断三个点的转向：若 cross(O, A, B) > 0，则 O->A->B 为逆时针（左转），说明 A 是凸包上的候选点；若 cross(O, A, B) ≤ 0，则为顺时针或共线，A 应被弹出。

Andrew 算法 (单调链)

Andrew 算法是最常用的凸包算法，也叫单调链（Monotone Chain）。核心思想是先按 x 坐标（x 相同按 y）排序，然后分别构建下凸壳和上凸壳。

算法流程

将所有点按 x 坐标排序（x 相同按 y 排序）
从左到右扫描，维护下凸壳：每次加入新点前，检查栈顶两个点与新点是否构成"左转"，若不是则弹出栈顶
从右到左扫描，维护上凸壳：同样的逻辑
合并下凸壳和上凸壳（去掉端点重复），得到完整凸包
  Andrew 算法分解

  下凸壳 (Lower Hull):        上凸壳 (Upper Hull):

        3                            6
       /                            / \
      2                            5   7
     /                            /
    1                            4
   /
  0

  从左到右建下凸壳              从右到左建上凸壳
  最后合并得到完整凸包: 0-1-2-3-5-6-7-4

代码实现

// Andrew 算法求凸包 (单调链)
// 返回凸包顶点，按逆时针排列
vector<Point> andrew(vector<Point> pts) {
    int n = pts.size();
    if (n <= 1) return pts;

    // 按 x 排序，x 相同按 y 排序
    sort(pts.begin(), pts.end(), [](const Point& a, const Point& b) {
        return a.x < b.x || (a.x == b.x && a.y < b.y);
    });

    vector<Point> hull;

    // 下凸壳
    for (int i = 0; i < n; i++) {
        while (hull.size() >= 2 &&
               cross(hull[hull.size()-2], hull.back(), pts[i]) <= 0)
            hull.pop_back();
        hull.push_back(pts[i]);
    }

    // 上凸壳
    int lowerSize = hull.size();
    for (int i = n - 2; i >= 0; i--) {
        while (hull.size() > lowerSize &&
               cross(hull[hull.size()-2], hull.back(), pts[i]) <= 0)
            hull.pop_back();
        hull.push_back(pts[i]);
    }

    // 去掉最后一个重复的起点
    hull.pop_back();
    return hull;
}
注意共线点的处理

上述代码中 cross(...) <= 0 会去掉凸包边上的共线点，只保留端点。如果需要保留凸包边上的点，改为 cross(...) < 0。根据题目要求选择。

时间复杂度 O(n log n)
空间复杂度 O(n)

Graham 扫描

Graham 扫描是经典的凸包算法。选择一个基准点（通常是最低最左的点），然后按极角排序其他点，依次扫描维护凸性。

算法流程

找到 y 坐标最小的点（y 相同取 x 最小）作为基准点 P0
将其他点按相对于 P0 的极角排序（极角相同按距离排序）
将前两个点入栈
依次处理每个点：检查栈顶两个点与当前点是否构成"左转"，若不是则弹出栈顶，直到满足条件后将当前点入栈
栈中即为凸包顶点
  Graham 扫描过程

  P0 是最低点，其余按极角排序

       P4 ---- P3
      /           \
    P5             P2
      \           /
       P6 ---- P1
        \      /
         \    /
          \  /
           P0 (基准点)

  扫描顺序: P0 -> P1 -> P2 -> P3 -> P4 -> P5 -> P6
  每步维护栈中点构成凸壳

极角排序

// 极角排序的比较函数
// 以 base 为基准，按极角从小到大排序
Point base;  // 基准点

bool polarCmp(const Point& a, const Point& b) {
    double c = cross(base, a, b);
    if (fabs(c) > eps) return c > 0;  // 极角小的在前
    return dist(base, a) < dist(base, b); // 极角相同，近的在前
}

代码实现

// Graham 扫描求凸包
vector<Point> graham(vector<Point> pts) {
    int n = pts.size();
    if (n <= 2) return pts;

    // 找基准点 (最低最左)
    int minIdx = 0;
    for (int i = 1; i < n; i++) {
        if (pts[i].y < pts[minIdx].y ||
            (pts[i].y == pts[minIdx].y && pts[i].x < pts[minIdx].x))
            minIdx = i;
    }
    swap(pts[0], pts[minIdx]);
    base = pts[0];

    // 极角排序 (跳过基准点)
    sort(pts.begin() + 1, pts.end(), polarCmp);

    // Graham 扫描
    vector<Point> stk;
    stk.push_back(pts[0]);
    stk.push_back(pts[1]);

    for (int i = 2; i < n; i++) {
        while (stk.size() >= 2 &&
               cross(stk[stk.size()-2], stk.back(), pts[i]) <= 0)
            stk.pop_back();
        stk.push_back(pts[i]);
    }

    return stk;
}

Andrew vs Graham

比较项Andrew (单调链)Graham 扫描
排序方式按 x 坐标排序按极角排序
实现难度更简单稍复杂
处理退化容易处理共线需要特殊处理
推荐程度首选经典但不如 Andrew 方便
实践建议

竞赛中推荐使用 Andrew 算法，代码更短且不易出错。Graham 扫描更适合理解凸包的思想，但在处理共线等退化情况时需要注意。

旋转卡壳 (Rotating Calipers)

旋转卡壳是一种在凸包上高效计算各种几何量的技术。基本思想是：用两条平行线"夹住"凸包，然后旋转这两条线，在旋转过程中维护目标量。

求凸包直径（最远点对）

凸包的直径是凸包上距离最远的两个点之间的距离。

  旋转卡壳求直径

     ___________
    /           \
   /    凸包     \
  |              |
   \            /
    \__________/

  对于凸包上的每条边 AB，
  找到离 AB 最远的点 P，
  |AP| 和 |BP| 中取较大值更新答案。
  随着边的旋转，最远点单调移动。

代码实现（求直径）

// 旋转卡壳求凸包直径 (最远点对距离)
double rotatingCalipers(vector<Point>& hull) {
    int n = hull.size();
    if (n == 2) return dist(hull[0], hull[1]);

    double ans = 0;
    int j = 1;  // 对踵点

    for (int i = 0; i < n; i++) {
        int ni = (i + 1) % n;
        // 旋转对踵点: 找到离边 hull[i]-hull[ni] 最远的点
        while (fabs(cross(hull[i], hull[ni], hull[(j+1)%n]))
             > fabs(cross(hull[i], hull[ni], hull[j]))) {
            j = (j + 1) % n;
        }
        // 更新答案
        ans = max(ans, max(dist(hull[i], hull[j]),
                            dist(hull[ni], hull[j])));
    }
    return ans;
}

求凸包宽度（最小宽度）

// 旋转卡壳求凸包宽度 (最小宽度)
double minWidth(vector<Point>& hull) {
    int n = hull.size();
    if (n == 2) return 0;

    double ans = 1e18;
    int j = 1;

    for (int i = 0; i < n; i++) {
        int ni = (i + 1) % n;
        while (fabs(cross(hull[i], hull[ni], hull[(j+1)%n]))
             > fabs(cross(hull[i], hull[ni], hull[j]))) {
            j = (j + 1) % n;
        }
        // 点 hull[j] 到边 hull[i]-hull[ni] 的距离
        ans = min(ans, pointToLine(hull[j], hull[i], hull[ni]));
    }
    return ans;
}

旋转卡壳的更多应用

应用说明复杂度
凸包直径最远点对距离O(n)
凸包宽度最小宽度（平行线距离）O(n)
两凸包最近距离两凸包间最近点对O(n + m)
两凸包最大距离两凸包间最远点对O(n + m)
凸包最小面积外接矩形一条边与凸包边重合O(n)
凸包三角剖分最大面积三角形三点在凸包上O(n^2)
旋转卡壳的关键观察

旋转卡壳的正确性依赖于凸包的凸性：随着一条边的旋转，对踵点（最远点）只会单调前进，不会回退。这保证了总时间复杂度为 O(n)。在实现时，通常用叉积大小来判断是否需要移动对踵点。

时间复杂度 O(n)

完整模板

#include <bits/stdc++.h>
using namespace std;

const double eps = 1e-9;

struct Point {
    double x, y;
    Point() : x(0), y(0) {}
    Point(double x, double y) : x(x), y(y) {}
    Point operator+(const Point& b) const { return {x+b.x, y+b.y}; }
    Point operator-(const Point& b) const { return {x-b.x, y-b.y}; }
    Point operator*(double k) const { return {x*k, y*k}; }
    bool operator==(const Point& b) const {
        return fabs(x-b.x)<eps && fabs(y-b.y)<eps;
    }
};

double cross(Point a, Point b) { return a.x*b.y - a.y*b.x; }
double cross(Point O, Point A, Point B) { return cross(A-O, B-O); }
double dot(Point a, Point b) { return a.x*b.x + a.y*b.y; }
double len(Point a) { return sqrt(dot(a,a)); }
double dist(Point a, Point b) { return len(a-b); }
double pointToLine(Point P, Point A, Point B) {
    return fabs(cross(B-A, P-A)) / len(B-A);
}

// ========== Andrew 凸包 ==========
vector<Point> andrew(vector<Point> pts) {
    int n = pts.size();
    if (n <= 1) return pts;
    sort(pts.begin(), pts.end(), [](Point& a, Point& b) {
        return a.x<b.x || (a.x==b.x && a.y<b.y);
    });
    vector<Point> hull;
    for (int i = 0; i < n; i++) {
        while (hull.size()>=2 &&
               cross(hull[hull.size()-2], hull.back(), pts[i]) <= 0)
            hull.pop_back();
        hull.push_back(pts[i]);
    }
    int low = hull.size();
    for (int i = n-2; i >= 0; i--) {
        while (hull.size()>low &&
               cross(hull[hull.size()-2], hull.back(), pts[i]) <= 0)
            hull.pop_back();
        hull.push_back(pts[i]);
    }
    hull.pop_back();
    return hull;
}

// ========== 旋转卡壳求直径 ==========
double diameter(vector<Point>& hull) {
    int n = hull.size();
    if (n == 1) return 0;
    if (n == 2) return dist(hull[0], hull[1]);
    double ans = 0;
    int j = 1;
    for (int i = 0; i < n; i++) {
        int ni = (i+1) % n;
        while (fabs(cross(hull[i], hull[ni], hull[(j+1)%n]))
             > fabs(cross(hull[i], hull[ni], hull[j])))
            j = (j+1) % n;
        ans = max(ans, max(dist(hull[i],hull[j]),
                            dist(hull[ni],hull[j])));
    }
    return ans;
}

// ========== 凸包面积 ==========
double hullArea(vector<Point>& hull) {
    double s = 0;
    int n = hull.size();
    for (int i = 0; i < n; i++)
        s += cross(hull[i], hull[(i+1)%n]);
    return fabs(s) / 2.0;
}

// ========== 凸包周长 ==========
double hullPerimeter(vector<Point>& hull) {
    double p = 0;
    int n = hull.size();
    for (int i = 0; i < n; i++)
        p += dist(hull[i], hull[(i+1)%n]);
    return p;
}

// ========== 判断点在凸包内 (二分) ==========
// 前提: hull 按逆时针排列, 以 hull[0] 为基准
bool inConvex(vector<Point>& hull, Point P) {
    int n = hull.size();
    if (cross(hull[0], hull[1], P) < 0) return false;
    if (cross(hull[0], hull[n-1], P) > 0) return false;
    int lo = 1, hi = n - 1;
    while (hi - lo > 1) {
        int mid = (lo + hi) / 2;
        if (cross(hull[0], hull[mid], P) >= 0) lo = mid;
        else hi = mid;
    }
    return cross(hull[lo], hull[hi], P) >= 0;
}

莫队概述

莫队算法（Mo's Algorithm）是一种离线处理区间查询的算法。核心思想是将所有查询按照特定顺序排序，使得相邻查询之间的区间变化很小，从而用较低的代价从一个查询转移到下一个查询。

核心思想

离线 + 排序 + 转移

1. 离线：将所有查询预先读入
2. 排序：按照 (左端点所在块, 右端点) 排序
3. 转移：维护当前区间 [L, R]，每次 O(1) 增删元素转移到下一个查询

适用条件

  • 查询是离线的（不要求强制在线）
  • 区间可以通过增加/删除端点来转移
  • 增加/删除一个元素的复杂度为 O(1)
  • 典型问题：区间内不同元素个数、区间众数、区间 mex 等
  莫队的核心：从一个查询转移到下一个查询

  当前查询 [L, R]    目标查询 [L', R']

  L ←───── L'         需要删除 L~L'-1 的元素
         R ─────→ R'   需要增加 R+1~R' 的元素

  每次转移的代价 = |L'-L| + |R'-R|
  通过巧妙排序，总转移代价为 O(n * sqrt(n))

复杂度分析

操作左指针移动右指针移动总复杂度
同一块内O(sqrt(n))O(n)O(n * sqrt(n))
跨块O(sqrt(n))O(n)O(n * sqrt(n))
总计q 个查询O((n+q) * sqrt(n))

普通莫队

普通莫队是最基本的形式。将序列分成 sqrt(n) 个块，查询按 (左端点所在块, 右端点) 排序。

分块与排序

int blockSize;  // 块大小，通常取 sqrt(n)

struct Query {
    int l, r, id;
};

// 莫队排序: 先按左端点所在块排，同一块内按右端点排
bool cmp(const Query& a, const Query& b) {
    int blockA = a.l / blockSize, blockB = b.l / blockSize;
    if (blockA != blockB) return blockA < blockB;
    // 奇偶性优化: 偶数块右端点升序，奇数块降序
    return (blockA & 1) ? (a.r > b.r) : (a.r < b.r);
}

完整流程

读入数组和所有查询，记录每个查询的 id
设置块大小 blockSize = sqrt(n)，对查询排序
初始化当前区间 [curL=0, curR=-1]（空区间）
依次处理每个查询：
  • 若 curL > q.l，则将 curL 左移，加入 arr[curL]
  • 若 curR < q.r，则将 curR 右移，加入 arr[curR]
  • 若 curL < q.l，则删除 arr[curL]，将 curL 右移
  • 若 curR > q.r，则删除 arr[curR]，将 curR 左移
记录当前区间的答案到 ans[q.id]

经典问题：区间不同元素个数

给定数组和若干查询 [l, r]，求每个查询区间内不同元素的个数。

#include <bits/stdc++.h>
using namespace std;

const int N = 300005;
int arr[N], cnt[N], ans[N], curAns;
int n, q, blockSize;

struct Query { int l, r, id; } queries[N];

bool cmp(const Query& a, const Query& b) {
    int ba = a.l / blockSize, bb = b.l / blockSize;
    if (ba != bb) return ba < bb;
    return (ba & 1) ? a.r > b.r : a.r < b.r;
}

void add(int pos) {
    cnt[arr[pos]]++;
    if (cnt[arr[pos]] == 1) curAns++;  // 新增一种元素
}

void remove(int pos) {
    cnt[arr[pos]]--;
    if (cnt[arr[pos]] == 0) curAns--;  // 减少一种元素
}

int main() {
    scanf("%d", &n);
    blockSize = sqrt(n);
    for (int i = 0; i < n; i++) scanf("%d", &arr[i]);

    scanf("%d", &q);
    for (int i = 0; i < q; i++) {
        scanf("%d%d", &queries[i].l, &queries[i].r);
        queries[i].l--; queries[i].r--;  // 转为 0-indexed
        queries[i].id = i;
    }

    sort(queries, queries + q, cmp);

    int curL = 0, curR = -1;
    for (int i = 0; i < q; i++) {
        int L = queries[i].l, R = queries[i].r;
        while (curL > L) add(--curL);
        while (curR < R) add(++curR);
        while (curL < L) remove(curL++);
        while (curR > R) remove(curR--);
        ans[queries[i].id] = curAns;
    }

    for (int i = 0; i < q; i++)
        printf("%d\n", ans[i]);
    return 0;
}

奇偶性优化

在排序时，对奇数块按右端点降序排列，偶数块按右端点升序排列。这样右指针在相邻块之间移动时呈"蛇形"走位，减少回退次数。

  奇偶性优化 (蛇形走位)

  块0(偶):  ──────────→
  块1(奇):  ←──────────
  块2(偶):  ──────────→
  块3(奇):  ←──────────

  右指针在块之间几乎不需要大幅回退
时间复杂度 O((n+q) * sqrt(n))

带修莫队

当查询区间的同时还有单点修改操作时，需要使用带修莫队。核心思想是在普通莫队的基础上增加一个时间维度。

三维莫队

将每个查询看作三维空间中的点 (l, r, t)，其中 t 表示该查询之前有多少次修改操作。

  带修莫队的三维排序

  维度: (左端点块, 右端点块, 时间)

  排序规则:
  1. 左端点所在块 (块大小 n^(2/3))
  2. 右端点所在块
  3. 时间戳

  块大小取 n^(2/3) 时总复杂度最优: O(n^(5/3))

块大小的选择

莫队类型块大小时间复杂度
普通莫队n^(1/2)O(n^(3/2))
带修莫队n^(2/3)O(n^(5/3))

代码实现

struct Query {
    int l, r, t, id;  // t: 该查询前的修改次数
};

struct Modify {
    int pos, oldVal, newVal;
};

int blockSize;

bool cmp(const Query& a, const Query& b) {
    int ba = a.l / blockSize, bb = b.l / blockSize;
    if (ba != bb) return ba < bb;
    int ra = a.r / blockSize, rb = b.r / blockSize;
    if (ra != rb) return ra < rb;
    return a.t < b.t;
}

// 应用第 t 次修改
void applyModify(int t, int curL, int curR) {
    int pos = mods[t].pos;
    // 如果修改的位置在当前区间内，先删除旧值
    if (curL <= pos && pos <= curR) {
        remove(pos);
    }
    // 应用修改
    arr[pos] = mods[t].newVal;
    // 如果修改的位置在当前区间内，加入新值
    if (curL <= pos && pos <= curR) {
        add(pos);
    }
}

// 撤销第 t 次修改
void undoModify(int t, int curL, int curR) {
    int pos = mods[t].pos;
    if (curL <= pos && pos <= curR) {
        remove(pos);
    }
    arr[pos] = mods[t].oldVal;
    if (curL <= pos && pos <= curR) {
        add(pos);
    }
}

void solve() {
    int curL = 0, curR = -1, curT = 0;

    for (int i = 0; i < q; i++) {
        // 先调整时间维度
        while (curT < queries[i].t)
            applyModify(++curT, curL, curR);
        while (curT > queries[i].t)
            undoModify(curT--, curL, curR);

        // 再调整空间维度 (同普通莫队)
        while (curL > queries[i].l) add(--curL);
        while (curR < queries[i].r) add(++curR);
        while (curL < queries[i].l) remove(curL++);
        while (curR > queries[i].r) remove(curR--);

        ans[queries[i].id] = curAns;
    }
}
注意事项

1. 修改操作需要记录 旧值和新值，以便撤销
2. 块大小取 pow(n, 2.0/3) 而非 sqrt(n)
3. 时间维度的调整要在空间维度之前进行

回滚莫队

回滚莫队用于add 容易但 delete 困难（或反过来）的问题。核心思想是只使用 add 操作，当右端点需要大幅回退时，暴力重建整个区间。

适用场景

  • 可以 O(1) add 但无法 O(1) delete（如维护最大值、区间众数等）
  • 需要维护的信息不支持撤销

算法流程

将查询按左端点所在块分组，每组内按右端点升序排列
对于同一块内的查询：
  • 右指针只增不减（因为按右端点排序）
  • 左指针每次从块的右边界出发，暴力右移到查询左端点
  • 处理完后，撤销左指针的所有操作（回滚）
跨块时，右指针需要重置到当前块的右边界
  回滚莫队的指针移动

  块边界  查询L  当前L
    |       |      |
    v       v      v
    ┌───────┬──────┐────────────────────→ R
    │  块内  │ 块外 │
    └───────┴──────┘
        ↑
        块右边界 blockEnd

  处理过程:
  1. R 从上次位置继续向右 (只增不减)
  2. L 从 blockEnd 向右暴力移动到查询 L
  3. 记录答案
  4. 撤销 L 的所有移动 (回滚到 blockEnd)

代码实现（区间最大值出现次数）

int blockSize;
int arr[N], cnt[N], curAns;

// 只支持 add，不支持 delete
void add(int pos) {
    cnt[arr[pos]]++;
    curAns = max(curAns, cnt[arr[pos]]);
}

// 回滚莫队
void rollbackMo(vector<Query>& queries) {
    sort(queries.begin(), queries.end(), [](Query& a, Query& b) {
        int ba = a.l / blockSize, bb = b.l / blockSize;
        if (ba != bb) return ba < bb;
        return a.r < b.r;  // 同块内右端点升序
    });

    for (int i = 0; i < queries.size(); ) {
        int block = queries[i].l / blockSize;
        int blockEnd = (block + 1) * blockSize - 1;
        int R = blockEnd;  // 右指针从块右边界开始

        // 处理同一块内的查询
        while (i < queries.size() &&
               queries[i].l / blockSize == block) {
            int L = queries[i].l, r = queries[i].r;

            // 右指针向右扩展 (只增不减)
            while (R < r) add(++R);

            // 保存当前状态
            int savedAns = curAns;

            // 左指针从块右边界暴力右移
            while (L > blockEnd) {
                add(L);  // 注意: 这里不能用 L--
                L++;
            }

            // 记录答案
            ans[queries[i].id] = curAns;

            // 回滚左指针的操作
            L = queries[i].l;
            while (L > blockEnd) {
                L--;
                cnt[arr[L]]--;
            }
            // 但不能直接回滚 cnt，需要重新计算
            // 更好的做法是使用备份数组
            curAns = savedAns;

            i++;
        }

        // 清空 cnt，为下一个块准备
        memset(cnt, 0, sizeof(cnt));
        curAns = 0;
    }
}
回滚莫队的变体

还有一种"不删除莫队"，适用于只能 delete 不能 add 的情况。此时左指针固定不动，右指针大幅回退时暴力重建。选择哪种取决于 add 和 delete 哪个更方便实现。

时间复杂度 O((n+q) * sqrt(n))

完整模板

普通莫队模板

#include <bits/stdc++.h>
using namespace std;

const int N = 1e5 + 5;
int arr[N], cnt[N], ans[N], curAns, blockSize;

struct Query { int l, r, id; } Q[N];

bool cmp(const Query& a, const Query& b) {
    int ba = a.l/blockSize, bb = b.l/blockSize;
    if (ba != bb) return ba < bb;
    return (ba&1) ? a.r > b.r : a.r < b.r;
}

void add(int pos) {
    // 根据题目修改
    cnt[arr[pos]]++;
    if (cnt[arr[pos]] == 1) curAns++;
}

void del(int pos) {
    // 根据题目修改
    cnt[arr[pos]]--;
    if (cnt[arr[pos]] == 0) curAns--;
}

int main() {
    int n, q;
    scanf("%d%d", &n, &q);
    blockSize = sqrt(n);
    for (int i = 0; i < n; i++) scanf("%d", &arr[i]);
    for (int i = 0; i < q; i++) {
        scanf("%d%d", &Q[i].l, &Q[i].r);
        Q[i].l--; Q[i].r--;
        Q[i].id = i;
    }
    sort(Q, Q+q, cmp);

    int L = 0, R = -1;
    for (int i = 0; i < q; i++) {
        while (L > Q[i].l) add(--L);
        while (R < Q[i].r) add(++R);
        while (L < Q[i].l) del(L++);
        while (R > Q[i].r) del(R--);
        ans[Q[i].id] = curAns;
    }

    for (int i = 0; i < q; i++)
        printf("%d\n", ans[i]);
}

带修莫队模板

#include <bits/stdc++.h>
using namespace std;

const int N = 1e5 + 5;
int arr[N], cnt[N], ans[N], curAns;
int n, q, blockSize, nq, nm;

struct Query { int l, r, t, id; } Q[N];
struct Modify { int pos, oldV, newV; } M[N];

bool cmp(const Query& a, const Query& b) {
    int ba = a.l/blockSize, bb = b.l/blockSize;
    if (ba != bb) return ba < bb;
    int ra = a.r/blockSize, rb = b.r/blockSize;
    if (ra != rb) return ra < rb;
    return a.t < b.t;
}

void add(int pos) { /* 根据题目修改 */ }
void del(int pos) { /* 根据题目修改 */ }

void apply(int t, int L, int R) {
    int p = M[t].pos;
    if (L <= p && p <= R) del(p);
    arr[p] = M[t].newV;
    if (L <= p && p <= R) add(p);
}

void undo(int t, int L, int R) {
    int p = M[t].pos;
    if (L <= p && p <= R) del(p);
    arr[p] = M[t].oldV;
    if (L <= p && p <= R) add(p);
}

int main() {
    scanf("%d%d", &n, &q);
    blockSize = pow(n, 2.0/3.0);
    for (int i = 0; i < n; i++) scanf("%d", &arr[i]);
    nq = nm = 0;
    while (q--) {
        char op; int x, y;
        scanf(" %c%d%d", &op, &x, &y);
        if (op == 'Q') {
            Q[nq] = {x-1, y-1, nm, nq};
            nq++;
        } else {
            M[nm] = {x-1, arr[x-1], y};
            arr[x-1] = y;
            nm++;
        }
    }
    // 恢复 arr 到初始状态 (需要额外保存)

    sort(Q, Q+nq, cmp);

    int L=0, R=-1, T=0;
    for (int i = 0; i < nq; i++) {
        while (T < Q[i].t) apply(++T, L, R);
        while (T > Q[i].t) undo(T--, L, R);
        while (L > Q[i].l) add(--L);
        while (R < Q[i].r) add(++R);
        while (L < Q[i].l) del(L++);
        while (R > Q[i].r) del(R--);
        ans[Q[i].id] = curAns;
    }

    for (int i = 0; i < nq; i++)
        printf("%d\n", ans[i]);
}

回滚莫队模板

#include <bits/stdc++.h>
using namespace std;

const int N = 1e5 + 5;
int arr[N], cnt[N], ans[N], curAns, blockSize;
int tmp[N]; // 用于回滚的备份

struct Query { int l, r, id; } Q[N];

void add(int pos) {
    cnt[arr[pos]]++;
    curAns = max(curAns, cnt[arr[pos]]);
}

int main() {
    int n, q;
    scanf("%d%d", &n, &q);
    blockSize = sqrt(n);
    for (int i = 0; i < n; i++) scanf("%d", &arr[i]);
    for (int i = 0; i < q; i++) {
        scanf("%d%d", &Q[i].l, &Q[i].r);
        Q[i].l--; Q[i].r--;
        Q[i].id = i;
    }

    sort(Q, Q+q, [](Query& a, Query& b) {
        int ba=a.l/blockSize, bb=b.l/blockSize;
        if(ba!=bb) return ba<bb;
        return a.r<b.r;
    });

    for (int i = 0; i < q; ) {
        int block = Q[i].l / blockSize;
        int R = (block+1) * blockSize - 1;
        curAns = 0;

        while (i < q && Q[i].l/blockSize == block) {
            while (R < Q[i].r) add(++R);

            int savedAns = curAns;
            int L = Q[i].l;
            while (L <= min(Q[i].r, R)) {
                add(L);
                L++;
            }
            ans[Q[i].id] = curAns;

            // 回滚
            while (L > Q[i].l) {
                L--;
                cnt[arr[L]]--;
            }
            curAns = savedAns;
            i++;
        }

        memset(cnt, 0, sizeof(cnt));
    }

    for (int i = 0; i < q; i++)
        printf("%d\n", ans[i]);
}

GCD 与 LCM

最大公约数 (GCD)

两个整数 a 和 b 的最大公约数 gcd(a, b) 是能同时整除 a 和 b 的最大正整数。欧几里得算法（辗转相除法）是求 GCD 的经典方法。

欧几里得算法原理

gcd(a, b) = gcd(b, a mod b)。当 b = 0 时，gcd(a, 0) = a。

时间复杂度: O(log(min(a, b)))

代码模板

// 方法一: 递归写法
long long gcd(long long a, long long b) {
    return b == 0 ? a : gcd(b, a % b);
}

// 方法二: STL __gcd (GCC)
// __gcd(a, b)  // 注意: 仅 GCC 可用

// 方法三: C++17 std::gcd
// #include <numeric>
// std::gcd(a, b)

// 方法四: 迭代写法 (避免栈溢出)
long long gcd_iter(long long a, long long b) {
    while (b) {
        a %= b;
        swap(a, b);
    }
    return a;
}C++

最小公倍数 (LCM)

两个整数 a 和 b 的最小公倍数 lcm(a, b) 是能被 a 和 b 整除的最小正整数。

公式

lcm(a, b) = a / gcd(a, b) * b

注意：先除后乘，防止溢出！即 a / gcd(a,b) * b 而非 a * b / gcd(a,b)。

long long lcm(long long a, long long b) {
    return a / gcd(a, b) * b; // 先除后乘防溢出
}C++

二进制 GCD (Stein's Algorithm)

一种不使用取模运算的 GCD 算法，利用位运算，在某些场景下比辗转相除更快。

// 二进制 GCD
long long gcd_stein(long long a, long long b) {
    if (a == 0) return b;
    if (b == 0) return a;
    int shift = __builtin_ctzll(a | b); // 共同因子 2 的个数
    a >>= __builtin_ctzll(a); // 去掉 a 的因子 2
    do {
        b >>= __builtin_ctzll(b); // 去掉 b 的因子 2
        if (a > b) swap(a, b);
        b -= a;
    } while (b);
    return a << shift;
}C++

GCD 的性质

性质 公式
交换律 gcd(a, b) = gcd(b, a)
结合律 gcd(a, gcd(b, c)) = gcd(gcd(a, b), c)
与 LCM 的关系 gcd(a, b) * lcm(a, b) = a * b
线性性质 gcd(a, b) = gcd(a, b + ka) 对任意整数 k
与差的关系 gcd(a, b) = gcd(a, b - a) = gcd(a, |a-b|)

扩展欧几里得算法 (Extended GCD)

扩展欧几里得算法（exGCD）不仅能求出 gcd(a, b)，还能找到一组整数 (x, y) 使得：

核心等式

ax + by = gcd(a, b)

时间复杂度: O(log(min(a, b)))

算法推导

当 b = 0 时，gcd(a, 0) = a，此时 x = 1, y = 0（因为 a*1 + 0*0 = a）
当 b ≠ 0 时，由 gcd(a, b) = gcd(b, a mod b)，先递归求出 bx' + (a mod b)y' = gcd(b, a mod b)
因为 a mod b = a - (a/b)*b，代入得 bx' + (a - (a/b)*b)y' = gcd
整理得 ay' + b(x' - (a/b)*y') = gcd
因此 x = y', y = x' - (a/b)*y'

代码模板

// 扩展欧几里得: 求 ax + by = gcd(a, b) 的一组解
// 返回值为 gcd(a, b)，x 和 y 通过引用返回
long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    long long d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;
}C++
理解递归中的 x, y 交换

注意代码中递归调用是 exgcd(b, a % b, y, x)，传入的 x 和 y 互换了位置。这是因为推导中 x = y', y = x' - (a/b)*y'，所以先让 x 接收递归返回的 y'（即新的 y），再计算 y = x' - (a/b)*y'（即原来的 x 减去修正项）。

通解形式

如果 (x0, y0) 是 ax + by = gcd(a, b) 的一组特解，则通解为：

x = x0 + k * (b / gcd(a, b))

y = y0 - k * (a / gcd(a, b))

其中 k 为任意整数。

裴蜀定理 (Bezout's Identity)

裴蜀定理是扩展欧几里得算法的理论基础，也是数论中最重要的定理之一。

裴蜀定理

对于任意整数 a 和 b，方程 ax + by = c 有整数解的充要条件是 gcd(a, b) | c（即 c 是 gcd(a, b) 的倍数）。

推论

  • ax + by = gcd(a, b) 一定有解（由 exGD 保证）
  • ax + by = 1 有解当且仅当 gcd(a, b) = 1（即 a, b 互素）
  • 若 gcd(a, b) = 1，则存在 x 使得 ax ≡ 1 (mod b)（即 a 在模 b 下有逆元）

应用：求解 ax + by = c

// 求解 ax + by = c 的一组解
// 返回 true 表示有解，false 表示无解
bool solve_linear(long long a, long long b, long long c,
                   long long &x, long long &y) {
    long long d = exgcd(a, b, x, y);
    if (c % d != 0) return false; // gcd(a,b) 不整除 c，无解
    // 先乘上倍数得到 ax + by = c 的特解
    x *= c / d;
    y *= c / d;
    // 可选: 调整到最小正整数解
    // x = ((x % (b/d)) + (b/d)) % (b/d);
    return true;
}C++

经典应用

POJ 1061 青蛙的约会

两只青蛙在同一条纬度线上跳跃，分别从位置 x 和 y 出发，每次分别跳 m 和 n 个单位，纬度总长为 L。问它们是否能相遇，若能则求最少跳跃次数。

// 分析: x + m*t ≡ y + n*t (mod L)
// 整理: (m-n)*t ≡ (y-x) (mod L)
// 即:   (m-n)*t + L*k = y-x
// 用 exgcd 求解即可
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (b == 0) { x = 1; y = 0; return a; }
    ll d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;
}

int main() {
    ll x, y, m, n, L;
    cin >> x >> y >> m >> n >> L;
    ll a = m - n, b = L, c = y - x;
    ll gx, gy;
    ll d = exgcd(a, b, gx, gy);
    if (c % d != 0) {
        cout << "Impossible" << endl;
    } else {
        gx *= c / d;
        long long mod = b / d;
        gx = ((gx % mod) + mod) % mod; // 调整到最小正整数
        cout << gx << endl;
    }
    return 0;
}C++

模逆元 (Modular Inverse)

模逆元是数论竞赛中最常用的工具之一。给定整数 a 和模数 p，a 在模 p 下的逆元 a-1 满足：

定义

a * a-1 ≡ 1 (mod p)

逆元存在的充要条件：gcd(a, p) = 1（即 a 和 p 互素）

为什么需要逆元？

在模运算中，除法不能直接取模：(a / b) % p ≠ (a % p) / (b % p)。但乘法可以取模。因此我们将除法转化为乘法：(a / b) % p = a * b-1 % p。

方法一：费马小定理

当 p 是素数时，由费马小定理 ap-1 ≡ 1 (mod p)，可得：

a-1 ≡ ap-2 (mod p) （p 为素数）

// 费马小定理求逆元 (p 必须是素数)
long long inv_fermat(long long a, long long p) {
    return qpow(a, p - 2, p);
}C++

方法二：扩展欧几里得

求 ax ≡ 1 (mod p) 等价于求 ax + py = 1 的解。用 exGD 求出 x 即为 a 的逆元。

// exGD 求逆元 (不要求 p 是素数，只要 gcd(a,p)=1)
long long inv_exgcd(long long a, long long p) {
    long long x, y;
    long long d = exgcd(a, p, x, y);
    if (d != 1) return -1; // 逆元不存在
    return ((x % p) + p) % p; // 调整到 [0, p) 范围
}C++
方法 前提条件 时间复杂度
费马小定理 p 必须是素数 O(log p)
扩展欧几里得 gcd(a, p) = 1 O(log p)
线性递推 p 是素数，预处理 [1, n] 的逆元 O(n) 预处理，O(1) 查询

线性求逆元

当需要求 [1, n] 内所有整数在模 p 下的逆元时，逐个用快速幂求解总复杂度为 O(n log p)，不够高效。线性递推法可以在 O(n) 时间内求出所有逆元。

时间复杂度: O(n)

递推公式推导

设 p = k*i + r，其中 k = p / i, r = p % i，则：

由 p = k*i + r，得 k*i + r ≡ 0 (mod p)
两边乘以 i-1 * r-1，得 k * r-1 + i-1 ≡ 0 (mod p)
整理得 i-1 ≡ -k * r-1 ≡ -(p/i) * (p%i)-1 (mod p)
即 inv[i] = -(p/i) * inv[p%i] (mod p)
等价写法: inv[i] = (p - p/i) * inv[p%i] % p
递推公式

inv[i] = (p - p / i) * inv[p % i] % p

初始条件：inv[1] = 1

代码模板

// 线性求 [1, n] 在模 p 下的所有逆元
const int MAXN = 3000005;
long long inv[MAXN];

void get_inv(int n, long long p) {
    inv[1] = 1;
    for (int i = 2; i <= n; i++)
        inv[i] = (p - p / i) * inv[p % i] % p;
}C++
前提条件

线性递推法要求 p 是素数，且 i < p。对于 p 不是素数的情况，需要使用 exGD 逐个求逆元。

线性求阶乘逆元

在组合数计算中，经常需要求阶乘的逆元。可以先用快速幂求 inv[n!]，再反向递推。

// 线性求阶乘逆元
long long fac[MAXN], inv_fac[MAXN];

void init_fac(int n, long long p) {
    fac[0] = 1;
    for (int i = 1; i <= n; i++)
        fac[i] = fac[i - 1] * i % p;
    // 用快速幂求 inv(n!)，然后反向递推
    inv_fac[n] = qpow(fac[n], p - 2, p);
    for (int i = n - 1; i >= 0; i--)
        inv_fac[i] = inv_fac[i + 1] * (i + 1) % p;
}C++

中国剩余定理 (CRT)

中国剩余定理用于求解一元线性同余方程组。这是中国古代数学的重要成就，最早由孙子在《孙子算经》中提出（"物不知数"问题）。

问题形式

求满足以下同余方程组的最小非负整数 x：

x ≡ a1 (mod m1)

x ≡ a2 (mod m2)

...

x ≡ an (mod mn)

其中 m1, m2, ..., mn 两两互素。

算法流程

计算 M = m1 * m2 * ... * mn
对每个 i，计算 Mi = M / mi
计算 Mi 在模 mi 下的逆元 ti（即 Mi * ti ≡ 1 mod mi）
合并：x = (a1*M1*t1 + a2*M2*t2 + ... + an*Mn*tn) mod M

代码模板

// 中国剩余定理 (CRT)
// a[] 为余数, m[] 为模数 (两两互素)
long long crt(long long a[], long long m[], int n) {
    long long M = 1;
    for (int i = 0; i < n; i++) M *= m[i];

    long long ans = 0;
    for (int i = 0; i < n; i++) {
        long long Mi = M / m[i];
        long long x, y;
        exgcd(Mi, m[i], x, y); // 求 Mi 的逆元
        ans = (ans + a[i] * Mi % M * ((x % m[i] + m[i]) % m[i])) % M;
    }
    return (ans % M + M) % M;
}C++

扩展中国剩余定理 (EXCRT)

当模数不两两互素时，使用扩展 CRT。核心思想是两两合并同余方程。

// 扩展 CRT: 模数不要求互素
// 合并 x ≡ a1 (mod m1) 和 x ≡ a2 (mod m2)
// 得到 x ≡ a (mod lcm(m1, m2))
typedef __int128 l128;

long long excrt(long long a[], long long m[], int n) {
    long long a1 = a[0], m1 = m[0];
    for (int i = 1; i < n; i++) {
        long long a2 = a[i], m2 = m[i];
        long long x, y;
        long long d = exgcd(m1, m2, x, y);
        if ((a2 - a1) % d != 0) return -1; // 无解
        // x * m1 + y * m2 = d
        // 需要的 k = (a2 - a1) / d * x
        long long t = m2 / d;
        x = ((l128)x * ((a2 - a1) / d) % t + t) % t;
        // 合并: a1 = a1 + x * m1, m1 = lcm(m1, m2)
        a1 = ((l128)a1 + (l128)x * m1 % ((l128)m1 * t)) % ((l128)m1 * t);
        m1 = m1 * t; // m1 = lcm(m1, m2)
    }
    return (a1 % m1 + m1) % m1;
}C++
CRT vs EXCRT

CRT 要求模数两两互素，直接套公式即可。EXCRT 不要求模数互素，但实现更复杂，需要两两合并。竞赛中遇到 CRT 问题时，先判断模数是否互素，再决定使用哪个模板。

完整 C++ 模板

模板一：GCD + exGCD + 逆元

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

// 快速幂
ll qpow(ll a, ll b, ll p) {
    ll res = 1; a %= p;
    while (b) {
        if (b & 1) res = (__int128)res * a % p;
        a = (__int128)a * a % p;
        b >>= 1;
    }
    return res;
}

// GCD
ll gcd(ll a, ll b) { return b ? gcd(b, a % b) : a; }
ll lcm(ll a, ll b) { return a / gcd(a, b) * b; }

// 扩展欧几里得
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (b == 0) { x = 1; y = 0; return a; }
    ll d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;
}

// 逆元 (费马小定理, p 为素数)
ll inv(ll a, ll p) { return qpow(a, p - 2, p); }

// 逆元 (exGD, 不要求 p 为素数)
ll inv2(ll a, ll p) {
    ll x, y;
    exgcd(a, p, x, y);
    return ((x % p) + p) % p;
}C++

模板二：线性求逆元 + 阶乘逆元

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

const int MAXN = 3000005;
ll inv[MAXN], fac[MAXN], inv_fac[MAXN];
ll p;

// 线性求 [1, n] 的逆元
void get_inv(int n) {
    inv[1] = 1;
    for (int i = 2; i <= n; i++)
        inv[i] = (p - p / i) * inv[p % i] % p;
}

// 阶乘及其逆元
ll qpow(ll a, ll b, ll mod) {
    ll res = 1; a %= mod;
    while (b) { if (b & 1) res = res * a % mod; a = a * a % mod; b >>= 1; }
    return res;
}

void init_fac(int n) {
    fac[0] = 1;
    for (int i = 1; i <= n; i++) fac[i] = fac[i-1] * i % p;
    inv_fac[n] = qpow(fac[n], p - 2, p);
    for (int i = n - 1; i >= 0; i--) inv_fac[i] = inv_fac[i+1] * (i+1) % p;
}C++

1. 字符串哈希原理

1.1 什么是字符串哈希

字符串哈希是一种将任意长度的字符串映射为一个固定长度整数的方法。如果两个字符串的哈希值不同，则它们一定不同；如果哈希值相同，则它们大概率相同（存在哈希冲突的可能）。

字符串哈希的核心优势在于：通过预处理，可以在 O(1) 时间内比较任意两个子串是否相等。

1.2 多项式滚动哈希（Polynomial Rolling Hash）

最常用的字符串哈希方法是多项式滚动哈希。对于字符串 S，其哈希值定义为：

多项式哈希公式

H(S) = (S[0] * B^(n-1) + S[1] * B^(n-2) + ... + S[n-1] * B^0) mod M

其中 B 是基数（base），M 是模数（mod），n 是字符串长度。

字符串: "abc"
哈希值: (a * B^2 + b * B^1 + c * B^0) mod M

其中 a=97, b=98, c=99 (ASCII码)
B = 131, M = 10^9 + 7

H("abc") = (97 * 131^2 + 98 * 131^1 + 99 * 131^0) mod M

1.3 滚动的思想

所谓"滚动"，是指当我们从左到右扫描字符串时，可以利用前一个位置的哈希值，通过一次乘法和一次加法快速计算出当前位置的哈希值，而不需要从头计算：

滚动递推公式

H[i] = (H[i-1] * B + S[i]) mod M

其中 H[i] 表示前缀 S[0..i] 的哈希值。

2. 哈希函数选择

2.1 基数 B 的选择

基数 B 应该大于字符集的大小。常用选择：

  • 小写字母：B = 131 或 B = 233
  • 大小写字母+数字：B = 131 或 B = 13331
  • 通用：B = 131
注意

B 不应太小（容易冲突），也不应太大（可能导致溢出）。选择一个质数作为 B 通常效果较好。

2.2 模数 M 的选择

模数 M 通常选择一个大质数。常用选择：

模数 M特点适用场景
10^9 + 7常用大质数一般场景
10^9 + 9另一个大质数双哈希中的第二个模数
2^64（自然溢出）利用 unsigned long long 自然溢出速度快，但理论上有被 hack 的风险
998244353NTT 友好质数需要配合 NTT 时

2.3 自然溢出

使用 unsigned long long 类型存储哈希值，利用 C++ 的自然溢出（相当于对 2^64 取模）。这种方法速度最快，但在竞赛中存在被构造数据 hack 的风险。如果题目允许，可以使用；如果是提交答案类题目，建议使用显式取模或双哈希。

// 自然溢出版本
typedef unsigned long long ull;
const ull B = 131;
vector<ull> buildHash(const string& s) {
    int n = s.size();
    vector<ull> h(n + 1, 0);
    for (int i = 0; i < n; i++)
        h[i + 1] = h[i] * B + s[i];  // 自动 mod 2^64
    return h;
}

3. 子串哈希O(1)查询

3.1 前缀哈希数组

预处理前缀哈希数组 H[] 和幂数组 powB[]：

前缀哈希定义

H[i] = 字符串 S 的前缀 S[0..i-1] 的哈希值（H[0] = 0）

H[i] = (H[i-1] * B + S[i-1]) mod M

3.2 子串哈希公式

要计算子串 S[l..r]（0-indexed，闭区间）的哈希值，可以利用前缀哈希的差值：

子串哈希公式

hash(S[l..r]) = H[r+1] - H[l] * B^(r-l+1) mod M

注意取模后可能出现负数，需要加 M 再取模：((H[r+1] - H[l] * powB[r-l+1]) % M + M) % M

前缀哈希数组:
H[0] = 0
H[1] = H[0]*B + S[0]              = hash("a")
H[2] = H[1]*B + S[1]              = hash("ab")
H[3] = H[2]*B + S[2]              = hash("abc")
H[4] = H[3]*B + S[3]              = hash("abcd")

子串哈希 hash(S[1..3]) = hash("bcd"):
= H[4] - H[1] * B^3
= hash("abcd") - hash("a") * B^3
= a*B^3 + b*B^2 + c*B^1 + d*B^0 - a*B^3
= b*B^2 + c*B^1 + d*B^0
= hash("bcd")  ✓

3.3 推导过程

为什么子串哈希公式成立？展开来看：

  • H[r+1] = S[0]*B^r + S[1]*B^(r-1) + ... + S[r]*B^0
  • H[l] * B^(r-l+1) = (S[0]*B^(l-1) + ... + S[l-1]*B^0) * B^(r-l+1)
  • 相减后，S[0] 到 S[l-1] 的项被完全抵消
  • 剩余部分恰好是 S[l]*B^(r-l) + S[l+1]*B^(r-l-1) + ... + S[r]*B^0，即子串 S[l..r] 的哈希值
// 子串哈希查询
ull getHash(const vector<ull>& h, const vector<ull>& pw,
            int l, int r) {
    // 返回 S[l..r] 的哈希值 (0-indexed)
    return h[r + 1] - h[l] * pw[r - l + 1];
}

4. 双哈希

4.1 为什么需要双哈希

单哈希存在哈希冲突的风险：两个不同的字符串可能恰好产生相同的哈希值。在竞赛中，出题人可能会构造专门的数据来卡单哈希。双哈希通过使用两个不同的哈希函数，将冲突概率降到极低。

双哈希策略

同时使用两组 (B, M) 计算两个哈希值。只有当两个哈希值都相等时，才认为两个字符串相等。

常用组合：(B1=131, M1=10^9+7) 和 (B2=13331, M2=10^9+9)

4.2 冲突概率分析

方法冲突概率安全性
单哈希 (mod 10^9+7)约 10^-9一般
单哈希 (自然溢出 2^64)约 10^-19（理论），但可被构造攻击不安全（竞赛中）
双哈希约 10^-18 到 10^-28非常安全

4.3 双哈希实现

struct DoubleHash {
    static const ull B1 = 131, B2 = 13331;
    static const ull M1 = 1e9 + 7, M2 = 1e9 + 9;

    vector<ull> h1, h2, pw1, pw2;

    DoubleHash(const string& s) {
        int n = s.size();
        h1.resize(n + 1, 0);
        h2.resize(n + 1, 0);
        pw1.resize(n + 1, 1);
        pw2.resize(n + 1, 1);
        for (int i = 0; i < n; i++) {
            h1[i + 1] = (h1[i] * B1 + s[i]) % M1;
            h2[i + 1] = (h2[i] * B2 + s[i]) % M2;
            pw1[i + 1] = pw1[i] * B1 % M1;
            pw2[i + 1] = pw2[i] * B2 % M2;
        }
    }

    // 返回双哈希对 (h1, h2)
    pair<ull, ull> get(int l, int r) {
        ull a = (h1[r + 1] - h1[l] * pw1[r - l + 1] % M1 + M1) % M1;
        ull b = (h2[r + 1] - h2[l] * pw2[r - l + 1] % M2 + M2) % M2;
        return {a, b};
    }

    // 比较两个子串是否相等
    bool equal(int l1, int r1, int l2, int r2) {
        return get(l1, r1) == get(l2, r2);
    }
};

5. 应用

5.1 最长公共子串

给定两个字符串 A 和 B，求它们的最长公共子串长度。

思路：二分答案 + 哈希。二分公共子串长度 L，将 A 的所有长度为 L 的子串哈希值存入 set，然后检查 B 是否有相同哈希值的子串。

// 最长公共子串 - 二分 + 哈希
int longestCommonSubstring(const string& a, const string& b) {
    StringHash ha(a), hb(b);
    int lo = 0, hi = min(a.size(), b.size());
    while (lo < hi) {
        int mid = (lo + hi + 1) / 2;
        set<ull> st;
        for (int i = 0; i + mid - 1 < (int)a.size(); i++)
            st.insert(ha.get(i, i + mid - 1));
        bool found = false;
        for (int i = 0; i + mid - 1 < (int)b.size(); i++) {
            if (st.count(hb.get(i, i + mid - 1))) {
                found = true;
                break;
            }
        }
        if (found) lo = mid;
        else hi = mid - 1;
    }
    return lo;
}

5.2 回文串判断

利用字符串哈希可以 O(1) 判断任意子串是否为回文串。维护正序哈希和逆序哈希，子串是回文串当且仅当其正序哈希值等于逆序哈希值。

// 回文串判断 - 双向哈希
struct PalinHash {
    StringHash fwd, rev;  // 正序和逆序哈希
    int n;

    PalinHash(const string& s) : n(s.size()) {
        fwd = StringHash(s);
        string r = s;
        reverse(r.begin(), r.end());
        rev = StringHash(r);
    }

    // 判断 S[l..r] 是否为回文串
    bool isPalin(int l, int r) {
        // 在逆序串中，S[l..r] 对应的位置是 [n-1-r, n-1-l]
        return fwd.get(l, r) == rev.get(n - 1 - r, n - 1 - l);
    }
};

5.3 字符串去重

给定 n 个字符串，统计有多少种不同的字符串。将每个字符串的哈希值存入 set，set 的大小即为答案。

5.4 子串出现次数

统计某个子串在文本串中出现的次数。可以将所有长度为 L 的子串哈希值存入 map，然后查询目标子串的哈希值出现次数。

5.5 最长回文子串（二分+哈希）

对每个位置，二分以该位置为中心的最长回文子串半径，用哈希 O(1) 判断是否回文。总复杂度 O(n log n)。

6. 完整C++模板

6.1 单哈希模板

#include <bits/stdc++.h>
using namespace std;

typedef unsigned long long ull;

struct StringHash {
    static const ull B = 131;
    vector<ull> h, pw;

    StringHash() {}
    StringHash(const string& s) { init(s); }

    void init(const string& s) {
        int n = s.size();
        h.assign(n + 1, 0);
        pw.assign(n + 1, 1);
        for (int i = 0; i < n; i++) {
            h[i + 1] = h[i] * B + s[i];
            pw[i + 1] = pw[i] * B;
        }
    }

    // S[l..r] 的哈希值 (0-indexed, 闭区间)
    ull get(int l, int r) const {
        return h[r + 1] - h[l] * pw[r - l + 1];
    }

    // 比较 S[l1..r1] 和 S[l2..r2]
    bool equal(int l1, int r1, int l2, int r2) const {
        return get(l1, r1) == get(l2, r2);
    }

    // 拼接两个子串的哈希: hash(S[l1..r1] + S[l2..r2])
    ull concat(int l1, int r1, int l2, int r2) const {
        return get(l1, r1) * pw[r2 - l2 + 1] + get(l2, r2);
    }
};

struct DoubleStringHash {
    static const ull B1 = 131, B2 = 13331;
    static const ull M1 = 1e9 + 7, M2 = 1e9 + 9;

    vector<ull> h1, h2, pw1, pw2;

    DoubleStringHash() {}
    DoubleStringHash(const string& s) { init(s); }

    void init(const string& s) {
        int n = s.size();
        h1.assign(n + 1, 0); h2.assign(n + 1, 0);
        pw1.assign(n + 1, 1); pw2.assign(n + 1, 1);
        for (int i = 0; i < n; i++) {
            h1[i + 1] = (h1[i] * B1 + s[i]) % M1;
            h2[i + 1] = (h2[i] * B2 + s[i]) % M2;
            pw1[i + 1] = pw1[i] * B1 % M1;
            pw2[i + 1] = pw2[i] * B2 % M2;
        }
    }

    pair<ull, ull> get(int l, int r) const {
        ull a = (h1[r + 1] - h1[l] * pw1[r - l + 1] % M1 + M1) % M1;
        ull b = (h2[r + 1] - h2[l] * pw2[r - l + 1] % M2 + M2) % M2;
        return {a, b};
    }

    bool equal(int l1, int r1, int l2, int r2) const {
        return get(l1, r1) == get(l2, r2);
    }
};
注意事项
  • 自然溢出版本不需要取模，但要注意使用 unsigned long long 类型
  • 显式取模版本在减法后要加 M 再取模，防止负数
  • 幂数组 pw[] 要预分配到与字符串长度相同的空间
  • 双哈希使用 pair 作为返回值，可以直接用 == 比较

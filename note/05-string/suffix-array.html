1. 后缀数组概述

后缀数组（Suffix Array, SA）是将一个字符串的所有后缀按字典序排序后得到的数组。它与 LCP（最长公共前缀）数组配合，可以高效解决大量字符串问题。

核心定义

对于长度为 n 的字符串 s，其后缀 s[i..n-1]（0 ≤ i < n）共有 n 个。

符号 含义
SA[i] 排名第 i 小的后缀的起始下标（i 从 0 开始）
rk[i]（或 Rank[i]） 以 i 开头的后缀的排名
lcp[i] s[SA[i-1]..] 与 s[SA[i]..] 的最长公共前缀长度

示例

字符串 s = "banana$"

后缀:               排序后:
  0: banana$         6: $
  1: anana$          5: a$
  2: nana$           3: ana$
  3: ana$            1: anana$
  4: na$             0: banana$
  5: a$              4: na$
  6: $               2: nana$

SA  = [6, 5, 3, 1, 0, 4, 2]
rk  = [4, 3, 6, 2, 5, 1, 0]
lcp = [0, 1, 3, 0, 0, 2, 0]  (lcp[0]无意义)

复杂度概览

倍增法 O(n log n) 排序 × log n = O(n log²n)
倍增法 + 基数排序 O(n log n)
SA-IS O(n)
Kasai 求 LCP O(n)

2. SA 构建 — 倍增法

倍增思想

倍增法的核心思想：先按每个字符排序，再按长度 2 排序，再按长度 4 排序...每次将排序关键字的长度翻倍，直到长度 ≥ n。利用上一轮的排名信息，用 双关键字排序（第一关键字为前半部分排名，第二关键字为后半部分排名）完成排序。

初始化（长度 1）：对每个字符排序，得到初始排名 rk[]。
倍增（长度 k → 2k）：对于以 i 开头、长度为 2k 的子串，第一关键字是 rk[i]（前 k 位排名），第二关键字是 rk[i+k]（后 k 位排名）。
双关键字排序：先按第二关键字排序，再按第一关键字稳定排序。使用 基数排序 可在 O(n) 内完成一轮。
重新编号：排序后对排名重新编号（去重），如果所有排名都唯一则提前终止。 
s = "aabaaaab"

初始 (k=1):
  字符:  a  a  b  a  a  a  a  b
  rk:    1  1  2  1  1  1  1  2

k=2 (按前1+后1排序):
  第一关键字: 1  1  2  1  1  1  1  2
  第二关键字: 1  2  1  1  1  1  2  0
  排名:       1  2  4  1  1  1  2  3

k=4 (按前2+后2排序):
  ...继续倍增直到 k >= n

每一轮排名越来越细，直到所有排名唯一

SA-IS 线性算法（了解）

SA-IS（Suffix Array by Induced Sorting）是一种 O(n) 的后缀数组构建算法。它将后缀分为 L型（递减）和 S型（递增），先递归处理部分后缀，再诱导排序出完整的 SA。虽然复杂度更优，但实现复杂，竞赛中通常使用倍增法即可。

实际建议

竞赛中 倍增法 + 基数排序（O(n log n)）是最常用的实现，代码简洁且常数小。只有在 n 特别大（如 10^7 级别）时才需要 SA-IS。

3. LCP 数组 — Kasai 算法

LCP 数组（Longest Common Prefix Array）定义：lcp[i] 是排名相邻的两个后缀 s[SA[i-1]..] 和 s[SA[i]..] 的最长公共前缀长度。

Kasai 算法

Kasai 算法利用一个巧妙的性质在线性时间内求出 LCP 数组：

关键观察

如果 lcp[rk[i]] > 0，那么 lcp[rk[i+1]] ≥ lcp[rk[i]] - 1。直观理解：如果以 i 开头的后缀与排名前一位的后缀有长度为 h 的公共前缀，那么以 i+1 开头的后缀与排名前一位的后缀至少有 h-1 的公共前缀。

初始化：令 h = 0，按原始下标 i = 0, 1, ..., n-1 依次处理。
计算 lcp[rk[i]]：如果 rk[i] == 0（排名最前），lcp[0] = 0，跳过。否则令 j = SA[rk[i]-1]（排名前一位的后缀起始位置），从 s[i+h] 和 s[j+h] 开始比较。
逐字符比较：当 s[i+h] == s[j+h] 时 h++。停止后 lcp[rk[i]] = h。
更新 h：h = max(0, h - 1)，为下一次迭代做准备。 
s = "aabaaaab", SA = [7,6,5,2,3,4,0,1]

i=0: rk[0]=6, j=SA[5]=4, s="aabaaaab" vs "aab"  -> lcp[6]=3, h=3
i=1: rk[1]=7, j=SA[6]=0, s="abaaaab"  vs "aabaaaab" -> lcp[7]=1, h=max(0,3-1)=1
i=2: rk[2]=3, j=SA[2]=5, s="baaaab"   vs "aab"  -> lcp[3]=0, h=0
i=3: rk[3]=4, j=SA[3]=2, s="aaaab"    vs "baaaab" -> lcp[4]=0, h=0
...

h 的"保底"性质保证了总比较次数不超过 2n

LCP 数组的性质

  • 任意两个后缀 s[SA[i]..] 和 s[SA[j]..]（i < j）的 LCP 等于 min(lcp[i+1], lcp[i+2], ..., lcp[j])（RMQ 问题）
  • 利用 ST 表可在 O(1) 内查询任意两个后缀的 LCP

4.1 最长公共子串（多个串）

4.2 不同子串个数

长度为 n 的字符串的不同子串总数 = n*(n+1)/2 - sum(lcp[i])。直观理解：每个后缀贡献 n - SA[i] 个新子串，但与排名前一位的后缀有 lcp[i] 个公共前缀要扣除。

// 不同子串个数
long long distinctSubstrings(int n) {
    long long ans = (long long)n * (n + 1) / 2;
    for (int i = 0; i < n; i++)
        ans -= lcp[i];
    return ans;
}

4.3 重复出现的子串

求至少出现 k 次的最长子串：在 LCP 数组上用长度为 k-1 的滑动窗口求窗口内 LCP 最小值的最大值（单调队列或二分+判定）。

4.4 字符串的最小循环节

利用后缀数组可以求字符串的最小周期。设 n - SA[0] 是最小后缀的长度（假设以特殊字符结尾），则最小周期可能与 LCP 和 SA 的性质有关。更常用的做法是直接用 KMP 的 next 数组。

4.5 求任意两个后缀的 LCP

用 Kasai 算法求出 LCP 数组后，配合 ST 表做 RMQ，可在 O(1) 时间内查询任意两个后缀的 LCP。

// ST 表预处理 LCP 数组，O(n log n) 预处理, O(1) 查询
int st[20][MAXN];
void buildST(int n) {
    for (int i = 0; i < n; i++) st[0][i] = lcp[i];
    for (int j = 1; (1 << j) <= n; j++)
        for (int i = 0; i + (1 << j) <= n; i++)
            st[j][i] = min(st[j - 1][i], st[j - 1][i + (1 << (j - 1))]);
}
int queryLCP(int i, int j) { // 求排名 i 和排名 j 的后缀的 LCP
    if (i == j) return n - SA[i];
    if (i > j) swap(i, j);
    i++; // lcp[i+1..j] 的最小值
    int k = __lg(j - i + 1);
    return min(st[k][i], st[k][j - (1 << k) + 1]);
}

5. C++ 完整模板

倍增法 + 基数排序 + Kasai

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1000010;

int sa[MAXN];     // 后缀数组
int rk[MAXN];     // 排名数组
int tmp[MAXN];    // 辅助数组（第二关键字排序结果）
int cnt[MAXN];    // 基数排序计数桶
int lcp[MAXN];    // LCP 数组
char s[MAXN];     // 原字符串（从下标 1 开始）
int n;            // 字符串长度

// 倍增法构建后缀数组，使用基数排序
void buildSA() {
    int m = 128; // 字符集大小

    // 第一轮：按单个字符排序
    for (int i = 1; i <= n; i++) cnt[rk[i] = s[i]]++;
    for (int i = 1; i <= m; i++) cnt[i] += cnt[i - 1];
    for (int i = n; i >= 1; i--) sa[cnt[rk[i]]--] = i;

    for (int w = 1; w < n; w <<= 1) { // w = 当前关键字长度
        int p = 0;

        // 第二关键字排序：先排 SA 中位置 > n-w 的（第二关键字为 0，排最前）
        for (int i = n - w + 1; i <= n; i++) tmp[++p] = i;
        for (int i = 1; i <= n; i++)
            if (sa[i] > w) tmp[++p] = sa[i] - w;

        // 第一关键字基数排序
        for (int i = 0; i <= m; i++) cnt[i] = 0;
        for (int i = 1; i <= n; i++) cnt[rk[tmp[i]]]++;
        for (int i = 1; i <= m; i++) cnt[i] += cnt[i - 1];
        for (int i = n; i >= 1; i--) sa[cnt[rk[tmp[i]]]--] = tmp[i];

        // 重新计算排名
        swap(rk, tmp);
        rk[sa[1]] = 1;
        p = 1;
        for (int i = 2; i <= n; i++) {
            // 判断是否与前一位的关键字对相同
            if (tmp[sa[i]] == tmp[sa[i - 1]] &&
                tmp[sa[i] + w] == tmp[sa[i - 1] + w])
                rk[sa[i]] = p;
            else
                rk[sa[i]] = ++p;
        }

        if (p == n) break; // 所有排名唯一，提前退出
        m = p;
    }
}

// Kasai 算法求 LCP 数组
void buildLCP() {
    int h = 0;
    for (int i = 1; i <= n; i++) {
        if (rk[i] == 1) { lcp[1] = 0; continue; }
        int j = sa[rk[i] - 1];
        while (i + h <= n && j + h <= n && s[i + h] == s[j + h]) h++;
        lcp[rk[i]] = h;
        if (h) h--;
    }
}

// 用法示例
int main() {
    scanf("%s", s + 1);
    n = strlen(s + 1);
    buildSA();
    buildLCP();

    // 输出后缀数组
    for (int i = 1; i <= n; i++)
        printf("SA[%d] = %d, LCP[%d] = %d\n", i, sa[i], i, lcp[i]);

    return 0;
}
注意事项

模板中字符串下标从 1 开始，末尾不加特殊字符。如果需要区分多个字符串，在拼接时使用不同的分隔符（如 $, #, % 等不出现在原串中的字符）。

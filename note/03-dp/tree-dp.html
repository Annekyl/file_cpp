1. 树形 DP 概述

树形 DP 是一类在树结构上进行动态规划的算法。核心思想是：以某个节点为根，将问题分解为若干子树上的子问题，利用 DFS 自底向上（从叶子到根）计算状态。

1.1 状态定义的一般形式

设 dp[u][...] 表示以 u 为根的子树中，满足某种条件的最优值。方括号内的维度取决于具体问题，例如：

  • dp[u] — 只与节点相关（如树的直径）
  • dp[u][0/1] — 节点选或不选（如最大独立集）
  • dp[u][k] — 子树中选 k 个节点（如树上背包）

1.2 计算流程

选定树根（通常为 1），建立邻接表
DFS 遍历，先递归处理所有子节点
回溯时，利用子节点的 dp 值更新当前节点
最终答案通常在 dp[root] 或所有节点的 dp 值中取最优

1.3 树形 DP 的计算顺序图示

          1 (根)
         / \
        2   3
       / \   \
      4   5   6

  DFS 访问顺序: 1 -> 2 -> 4 -> 5 -> 3 -> 6
  DP 计算顺序: 4 -> 5 -> 2 -> 6 -> 3 -> 1  (后序遍历)

  即: 先计算叶子节点, 再逐步向上合并到根
核心要点

树形 DP 的本质是后序遍历：每个节点在处理完所有子树之后才进行自身的状态转移。这保证了子问题的正确性。

2. 树的直径

树的直径定义为树上任意两点之间最长路径的长度。这是一个经典的树形 DP 问题，也有两次 DFS 的解法。

2.1 DP 解法

状态定义

dp[u] = 从节点 u 出发，向子树方向走的最长路径长度（即 u 到其子树中某个叶子的最长距离）。

状态转移

对于节点 u 的每个子节点 v（边权为 w）：

// 转移方程
dp[u] = max(dp[u], dp[v] + w)

// 直径的候选值: 经过 u 的最长路径 = 最长子树路径 + 次长子树路径
diameter = max(diameter, best1 + best2)
// 其中 best1, best2 是 dp[v] + w 的最大和次大值

计算过程图示

            1
           / \
         (2) (3)
         /     \
        2       3
       / \       \
     (1) (4)    (2)
     /     \      \
    4       5      6

  dp[4] = 0          (叶子)
  dp[5] = 0          (叶子)
  dp[2] = max(0+1, 0+4) = 4    经过2: 1+4=5
  dp[6] = 0          (叶子)
  dp[3] = 0+2 = 2
  dp[1] = max(4+2, 2+3) = 6    经过1: (4+2)+(2+3) = 11

  直径 = max(5, 11) = 11   (路径: 5->2->1->3->6, 权值 4+2+3+2)

C++ 代码模板

#include <bits/stdc++.h>
using namespace std;

const int N = 1e5 + 5;
vector<pair<int,int>> g[N];  // {邻接点, 边权}
int dp[N];      // dp[u]: u 向下走的最长路径
int diameter;   // 全局直径

void dfs(int u, int fa) {
    dp[u] = 0;
    int best1 = 0, best2 = 0;  // 最长 & 次长
    (auto& [v, w] : g[u]) {
        if (v == fa) continue;
        dfs(v, u);
        int len = dp[v] + w;
        dp[u] = max(dp[u], len);
        // 更新最长和次长
        if (len >= best1) {
            best2 = best1;
            best1 = len;
        } else if (len > best2) {
            best2 = len;
        }
    }
    diameter = max(diameter, best1 + best2);
}

int main() {
    int n;
    cin >> n;
    for (int i = 1; i < n; i++) {
        int u, v, w;
        cin >> u >> v >> w;
        g[u].push_back({v, w});
        g[v].push_back({u, w});
    }
    diameter = 0;
    dfs(1, 0);
    cout << diameter << endl;
    return 0;
}
时间复杂度 O(n)
空间复杂度 O(n)

2.2 两次 DFS 解法

另一种经典做法：

  1. 从任意节点（如 1）出发 DFS，找到距离最远的节点 x
  2. 从 x 出发 DFS，找到距离最远的节点 y
  3. x 到 y 的距离即为直径
int dist[N], farNode;
int maxDist;

void dfsDist(int u, int fa, int d) {
    dist[u] = d;
    if (d > maxDist) {
        maxDist = d;
        farNode = u;
    }
    for (auto& [v, w] : g[u]) {
        if (v == fa) continue;
        dfsDist(v, u, d + w);
    }
}

int getDiameter() {
    maxDist = -1;
    dfsDist(1, 0, 0);  // 第一次: 找到离 1 最远的 farNode
    maxDist = -1;
    dfsDist(farNode, 0, 0);  // 第二次: 从 farNode 出发
    return maxDist;
}

2.3 两种方法的比较

方法 优点 缺点 适用场景
树形 DP 一次 DFS，同时可求每个节点的最远距离 代码稍长 需要子树信息、带权树
两次 DFS 代码简洁，直觉清晰 两次遍历，只能求直径长度 只需直径长度
经典问题

Luogu P2195 - HXY造公园 — 树的直径 + 并查集

Luogu P1099 - 树网的核 — 树的直径 + 暴力/枚举

3. 树的重心

树的重心（Centroid）是树中一个特殊节点：删除该节点后，剩余各个连通块的大小的最大值最小。

3.1 性质

  • 一棵树的重心有 1 个或 2 个
  • 若有两个重心，则它们必然相邻，且删除后各自所在连通块大小恰好为 n/2
  • 重心到所有点的距离之和最小
  • 两棵树合并时，新树的重心在原来两棵树重心的路径上

3.2 状态定义

sz[u] = 以 u 为根的子树大小

对于节点 u，删除它后：

  • 每个子树 v 的连通块大小为 sz[v]
  • "上方"连通块大小为 n - sz[u]

重心是使 max(max_subtree_size, n - sz[u]) 最小的节点。

3.3 计算图示

          1            sz: 7
         / \
        2   3          sz: 3, 3
       / \   \
      4   5   6        sz: 1, 1, 1
          |
          7            sz: 1

  对于节点 1:
    子树最大 = max(sz[2], sz[3]) = max(3, 3) = 3
    上方    = n - sz[1] = 7 - 7 = 0
    max_size = 3

  对于节点 2:
    子树最大 = max(sz[4], sz[5]) = max(1, 2) = 2
    上方    = 7 - 3 = 4
    max_size = 4

  对于节点 5:
    子树最大 = sz[7] = 1
    上方    = 7 - 2 = 5
    max_size = 5

  重心 = 节点 1 或 节点 3 (max_size = 3 最小)

3.4 C++ 代码模板

#include <bits/stdc++.h>
using namespace std;

const int N = 1e5 + 5;
vector<int> g[N];
int sz[N], n;
int centroid, minMaxSize = INT_MAX;

void dfs(int u, int fa) {
    sz[u] = 1;
    int maxSub = 0;  // 最大子树大小
    for (int v : g[u]) {
        if (v == fa) continue;
        dfs(v, u);
        sz[u] += sz[v];
        maxSub = max(maxSub, sz[v]);
    }
    // "上方"连通块
    int upSize = n - sz[u];
    maxSub = max(maxSub, upSize);
    // 更新重心
    if (maxSub < minMaxSize) {
        minMaxSize = maxSub;
        centroid = u;
    }
}

int main() {
    cin >> n;
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u);
    }
    dfs(1, 0);
    cout << centroid << endl;
    return 0;
}
时间复杂度 O(n)
经典问题

Luogu P1364 - 医院设置 — 求带权重心使距离和最小

Luogu P2986 - [USACO10MAR] Great Cow Gathering G — 换根DP求距离和

4. 树上最大独立集

给定一棵树，每个节点有权值。选出若干节点使得任意两个被选节点不相邻，且权值之和最大。这是树上"选与不选"类型的经典模型。

4.1 状态定义

  • dp[u][1] = 选节点 u 时，以 u 为根的子树的最大独立集权值和
  • dp[u][0] = 不选节点 u 时，以 u 为根的子树的最大独立集权值和

4.2 状态转移

// 选 u: 子节点 v 不能选
dp[u][1] = val[u] + sum(dp[v][0])

// 不选 u: 子节点 v 可选可不选, 取较大值
dp[u][0] = sum(max(dp[v][0], dp[v][1]))

4.3 计算图示

  树结构 (节点旁数字为权值):

            1(10)
           / \
        2(5)  3(8)
       / \
    4(3)  5(7)

  自底向上计算:
  dp[4][0]=0,  dp[4][1]=3
  dp[5][0]=0,  dp[5][1]=7

  dp[2][1] = 5 + dp[4][0] + dp[5][0] = 5+0+0 = 5
  dp[2][0] = max(0,3) + max(0,7) = 3+7 = 10

  dp[3][0]=0,  dp[3][1]=8

  dp[1][1] = 10 + dp[2][0] + dp[3][0] = 10+10+0 = 20
  dp[1][0] = max(dp[2][0],dp[2][1]) + max(dp[3][0],dp[3][1])
           = max(10,5) + max(0,8) = 10+8 = 18

  答案 = max(dp[1][0], dp[1][1]) = max(18, 20) = 20
  选中节点: 1, 4, 5 (权值 10+3+7=20)

4.4 C++ 代码模板

#include <bits/stdc++.h>
using namespace std;

const int N = 1e5 + 5;
vector<int> g[N];
int val[N], dp[N][2];

void dfs(int u, int fa) {
    dp[u][1] = val[u];
    dp[u][0] = 0;
    for (int v : g[u]) {
        if (v == fa) continue;
        dfs(v, u);
        dp[u][1] += dp[v][0];
        dp[u][0] += max(dp[v][0], dp[v][1]);
    }
}

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> val[i];
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u);
    }
    dfs(1, 0);
    cout << max(dp[1][0], dp[1][1]) << endl;
    return 0;
}
时间复杂度 O(n)
空间复杂度 O(n)
扩展: 无权最大独立集

若每个节点权值均为 1，则问题退化为"在树上选最多的互不相邻节点"，也即经典的树上最大独立集。转移方程不变，只是权值恒为 1。

5. 树上背包 (分组背包)

给定一棵 n 个节点的有根树，每个节点有体积和价值。在以 u 为根的子树中恰好选择 k 个节点（包含 u），求最大价值。这是经典的"树上分组背包"问题。

5.1 状态定义

dp[u][k] = 在以 u 为根的子树中，选择恰好 k 个节点（必含 u）时的最大价值。

5.2 转移思想

将 u 的每个子节点 v 看作一组物品，从这组物品中选若干个（0 ~ sz[v]）。这是一个分组背包，需要逆序枚举总容量以保证 01 背包性质。

// 对于 u 的子节点 v, 枚举分配给 v 子树的节点数 j
// 必须倒序枚举 k, 保证 v 子树的选择只被用一次
for (int i = 0; i < g[u].size(); i++) {
    int v = g[u][i];
    dfs(v, u);
    for (int k = curSize; k >= 1; k--)       // 倒序枚举总容量
        for (int j = 1; j <= min(k, sz[v]); j++) // 分配 j 个给 v
            dp[u][k] = max(dp[u][k], dp[u][k-j] + dp[v][j]);
}

5.3 计算图示

  以节点 1 为根, 选恰好 2 个节点:

          1 (val=5)
         / \
      2(3)  3(4)
       |
     4(2)

  初始: dp[1] = [5, -inf, -inf, -inf]  (只选1个即自身)

  处理子节点 2 (dp[2] = [3, -inf], sz[2]=2):
    k 从 2 倒序到 1:
    k=2: dp[1][2] = max(dp[1][2], dp[1][1]+dp[2][1]) = max(-inf, 5+3) = 8
    k=1: 不变

  处理子节点 3 (dp[3] = [4], sz[3]=1):
    k=2: dp[1][2] = max(8, dp[1][1]+dp[3][1]) = max(8, 5+4) = 9
    k=1: 不变

  dp[1][2] = 9  (选节点 1 和 3)

5.4 C++ 代码模板

#include <bits/stdc++.h>
using namespace std;

const int N = 205;
vector<int> g[N];
int val[N], sz[N];
int dp[N][N];  // dp[u][k]: 子树 u 中选 k 个节点的最大价值
int n, m;

void dfs(int u, int fa) {
    sz[u] = 1;
    // 初始化: 只选自己
    dp[u][1] = val[u];

    for (int v : g[u]) {
        if (v == fa) continue;
        dfs(v, u);

        // 分组背包: 倒序枚举总容量
        for (int k = sz[u]; k >= 1; k--) {
            for (int j = 1; j <= min(sz[v], m - k); j++) {
                dp[u][k + j] = max(dp[u][k + j], dp[u][k] + dp[v][j]);
            }
        }
        sz[u] += sz[v];
    }
}

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> val[i];
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u);
    }

    // memset dp 为极小值, dp[u][0] = 0
    memset(dp, 0xcf, sizeof(dp));
    for (int i = 1; i <= n; i++) dp[i][0] = 0;

    dfs(1, 0);
    cout << dp[1][m] << endl;
    return 0;
}
时间复杂度 O(n * m^2) 朴素 / O(n * m) 求和优化
复杂度优化提示

朴素的树上背包复杂度为 O(n * m^2)。可以通过限制内层循环的上界（只枚举到当前已合并的子树大小之和）来优化到 O(n * m)，这相当于对每条边只贡献 O(min(sz[u], m)) 的转移量，总复杂度为 O(n * m)。

经典问题

Luogu P2014 - [CTSC1997] 选课 — 树上背包模板题，森林加超级根

CF 1092F - Tree with Maximum Cost — 树上背包变体

6. 换根 DP (Rerooting)

换根 DP 是树形 DP 的高级技巧。当题目要求以每个节点为根时的最优值，如果对每个节点都做一次 DFS，复杂度为 O(n^2)。换根 DP 通过两次 DFS（一次自底向上、一次自顶向下）将复杂度降为 O(n)。

6.1 什么时候使用换根 DP

  • 需要求每个节点作为根时的答案（而非只求某一个根的答案）
  • 问题可以拆分为"向下的贡献"和"向上的贡献"
  • 典型场景：每个节点到其他所有节点的距离之和、每个节点的子树之外的信息

6.2 两次 DFS 模板

第一遍 DFS (自底向上)

任选一个根（通常为 1），计算每个节点只考虑子树时的 dp 值。

第二遍 DFS (自顶向下)

从根出发，利用父节点的 dp 值，推算出"将根从父节点换到子节点"后的值。

// down[u]: 自底向上, 只考虑 u 的子树
// dp[u]:   最终答案, 考虑整棵树以 u 为根
//
// 第一遍: down[u] 从子节点 v 的 down[v] 转移
// 第二遍: dp[v] = f(dp[u], down[v])
//         即 v 的答案 = u 的答案 减去 v 对 u 的贡献, 再加上 u 对 v 的新贡献

6.3 经典例题: 树上距离之和

给定一棵 n 个节点的树，对每个节点 u，求 u 到其他所有节点的距离之和。

状态定义

  • down[u] = 以 1 为根时，u 到其子树中所有节点的距离之和
  • sz[u] = 以 1 为根时，u 的子树大小
  • dp[u] = 以 u 为根时，u 到所有其他节点的距离之和（最终答案）

转移方程

// 第一遍 (自底向上)
down[u] = sum(down[v] + sz[v])  // v 是 u 的子节点
sz[u] = 1 + sum(sz[v])

// 第二遍 (自顶向下): 换根 u -> v
// 当根从 u 换到 v 时:
//   - v 子树中的节点距离各减少 1
//   - 其余节点 (n - sz[v] 个) 距离各增加 1
dp[v] = dp[u] - sz[v] + (n - sz[v])
dp[v] = dp[u] + n - 2 * sz[v]

换根过程图示

  树结构:
        1
       / \
      2   3
     /
    4

  n = 4

  === 第一遍: 以 1 为根, 自底向上 ===
  sz[4]=1, down[4]=0
  sz[2]=2, down[2]=(down[4]+sz[4]) + 0 = 1
  sz[3]=1, down[3]=0
  sz[1]=4, down[1]=(down[2]+sz[2]) + (down[3]+sz[3]) = (1+2) + (0+1) = 4

  dp[1] = down[1] = 4

  === 第二遍: 自顶向下, 换根 ===
  根从 1 换到 2:
    dp[2] = dp[1] + n - 2*sz[2] = 4 + 4 - 4 = 4
  根从 1 换到 3:
    dp[3] = dp[1] + n - 2*sz[3] = 4 + 4 - 2 = 6
  根从 2 换到 4:
    dp[4] = dp[2] + n - 2*sz[4] = 4 + 4 - 2 = 6

  答案: dp = [4, 4, 6, 6]
  验证: 节点1到其余: 1+1+2=4  节点2: 1+2+1=4  节点3: 1+2+3=6  节点4: 2+1+3=6  OK!

6.4 C++ 完整代码

#include <bits/stdc++.h>
using namespace std;

const int N = 2e5 + 5;
vector<int> g[N];
long long down[N];  // down[u]: u 到子树中所有节点的距离和
long long dp[N];    // dp[u]: 以 u 为根时到所有节点的距离和
int sz[N];
int n;

// 第一遍 DFS: 自底向上计算 down[u] 和 sz[u]
void dfs1(int u, int fa) {
    sz[u] = 1;
    down[u] = 0;
    for (int v : g[u]) {
        if (v == fa) continue;
        dfs1(v, u);
        sz[u] += sz[v];
        down[u] += down[v] + sz[v];
    }
}

// 第二遍 DFS: 自顶向下, 利用父节点信息推算 dp[v]
void dfs2(int u, int fa) {
    for (int v : g[u]) {
        if (v == fa) continue;
        // 换根: 将根从 u 换到 v
        // v 子树内: 距离减少 sz[v]
        // v 子树外: 距离增加 (n - sz[v])
        dp[v] = dp[u] + n - 2 * sz[v];
        dfs2(v, u);
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cin >> n;
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u);
    }
    dfs1(1, 0);
    dp[1] = down[1];
    dfs2(1, 0);

    for (int i = 1; i <= n; i++)
        cout << dp[i] << (i == n ? '\n' : ' ');
    return 0;
}
时间复杂度 O(n)
空间复杂度 O(n)

6.5 换根 DP 通用模板

换根 DP 的核心在于"去除旧贡献、加入新贡献"。以下是通用模板框架：

// 换根 DP 通用框架 (适用于需要以每个节点为根求答案的场景)
//
// 第一遍 dfs_down: 计算每棵子树对父节点的贡献
// 第二遍 dfs_up:   计算"子树以外的部分"对当前节点的贡献

void dfs_down(int u, int fa) {
    down[u] = combine(u 的初始值);
    for (int v : g[u]) {
        if (v == fa) continue;
        dfs_down(v, u);
        down[u] = merge(down[u], contribution(v, down[v]));
    }
}

void dfs_up(int u, int fa, T up_contrib) {
    // dp[u] = merge(down[u], up_contrib)
    dp[u] = merge(down[u], up_contrib);

    // 计算前缀和 & 后缀和, 以便快速去除某个子节点的贡献
    int deg = g[u].size();
    vector<T> pre(deg + 2), suf(deg + 2);
    pre[0] = identity();
    for (int i = 0; i < deg; i++) {
        int v = g[u][i];
        pre[i + 1] = merge(pre[i], (v != fa ? contribution(v, down[v]) : identity()));
    }
    suf[deg] = identity();
    for (int i = deg - 1; i >= 0; i--) {
        int v = g[u][i];
        suf[i] = merge(suf[i + 1], (v != fa ? contribution(v, down[v]) : identity()));
    }

    for (int i = 0; i < deg; i++) {
        int v = g[u][i];
        if (v == fa) continue;
        // v 的上方贡献 = up_contrib + 所有兄弟子树的贡献
        T new_up = merge(up_contrib, merge(pre[i], suf[i + 1]));
        dfs_up(v, u, addParentContribution(u, new_up));
    }
}
前缀和 / 后缀和技巧

在换根时，需要从 u 的所有子节点中"去掉"v 的贡献。如果直接在循环中减去，需要额外处理。使用前缀和 + 后缀和可以在 O(deg(u)) 的时间内计算出去掉任意一个子节点后的合并结果，避免嵌套循环。

总结

算法 状态定义 转移方式 复杂度
树的直径 (DP) dp[u]: u 向下最长路径 最长 + 次长合并 O(n)
树的直径 (2x DFS) dist[u] 两次 DFS 求最远 O(n)
树的重心 sz[u] 子树大小取 max O(n)
最大独立集 dp[u][0/1] 选与不选分类转移 O(n)
树上背包 dp[u][k] 分组背包 + 倒序枚举 O(nm)
换根 DP down[u], dp[u] 两遍 DFS: 子树 + 补集 O(n)
解题技巧总结
  • 选根: 通常以 1 为根即可; 如果是无根树且需要以每个节点为根, 考虑换根 DP
  • 状态设计: 先确定 dp[u] 表示什么, 再考虑如何从子节点转移
  • 初始化: 叶节点的 dp 值是递归的终止条件, 要仔细处理
  • 树上背包: 注意倒序枚举, 控制合并的循环上界来优化复杂度
  • 换根: 关键公式 dp[v] = dp[u] - contribution(v) + new_contribution
  • 空间: 树形 DP 通常只需 O(n) 空间, 但树上背包需要 O(nm)

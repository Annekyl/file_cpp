1. 区间 DP 概述

什么是区间 DP

区间 DP 是一类在 区间 上进行状态转移的动态规划。核心思想是：将一个大区间的问题分解为若干小区间的子问题，通过枚举 分割点 来合并子问题的答案。

区间 DP 适用于满足以下特征的问题：

  • 问题定义在一段 连续区间 上
  • 大区间的答案可以由 两个或多个不相交的小区间 合并得到
  • 满足 最优子结构 和 无后效性

状态定义

定义二维状态数组：

核心状态

dp[l][r] 表示将区间 [l, r] 合并（或处理）为一个整体的最小（最大）代价。

转移方程的一般形式

// 枚举分割点 k，将 [l,r] 分为 [l,k] 和 [k+1,r]
for (int k = l; k < r; k++) {
    dp[l][r] = min(dp[l][r], dp[l][k] + dp[k+1][r] + cost(l, k, r));
}

其中 cost(l, k, r) 是将两段子区间合并为 [l, r] 时产生的额外代价。不同题目中这个代价的定义不同。

DP 表的填充顺序

区间 DP 的关键在于填充顺序：先填小区间，再填大区间。通常按 区间长度 从小到大枚举：

for (int len = 2; len <= n; len++)       // 枚举区间长度
    for (int l = 1; l + len - 1 <= n; l++)  // 枚举左端点
    {
        int r = l + len - 1;               // 计算右端点
        for (int k = l; k < r; k++)         // 枚举分割点
            dp[l][r] = min(dp[l][r], dp[l][k] + dp[k+1][r] + cost(l, k, r));
    }

  填充顺序:  对角线1(长度1) -> 对角线2(长度2) -> ... -> 对角线n(长度n)

  (n) = 长度 >= 2的区间, 后计算

复杂度分析

时间复杂度 O(n3)
空间复杂度 O(n2)

状态数 O(n2)，每个状态枚举分割点 O(n)，总计 O(n3)。空间为二维数组 O(n2)。

技巧提示

区间 DP 的代码框架高度统一，几乎所有的区间 DP 问题都可以用 三层循环 的模板来解决。关键是正确地定义状态和转移方程。

2. 经典模型 - 石子合并

石子合并 (Stone Merge)

有 n 堆石子排成一排，每次可以将 相邻的两堆 合并为一堆，合并的代价为两堆石子的总数。求将所有石子合并为一堆的 最小总代价。

输入：第一行 n，第二行 n 个整数表示每堆石子数量。

输出：最小总代价。

状态定义

状态

dp[l][r] = 将区间 [l, r] 内的所有石子合并为一堆的最小代价。

sum[i] = 石子数量的前缀和，用于 O(1) 计算区间和。

转移方程

对于区间 [l, r]，枚举最后一个分割点 k（即最后将 [l, k] 和 [k+1, r] 合并）：

// 前缀和: sum[i] = a[1] + a[2] + ... + a[i]
// 区间和: sum[r] - sum[l-1]

dp[l][r] = min( dp[l][k] + dp[k+1][r] + sum[r] - sum[l-1] )   // l <= k < r

// 边界: dp[i][i] = 0 （单堆不需要合并）
// 答案: dp[1][n]

完整 C++ 代码

#include <bits/stdc++.h>
using namespace std;

const int INF = 0x3f3f3f3f;
const int N = 310;

int n;
int a[N], sum[N];
int dp[N][N];

int main() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        scanf("%d", &a[i]);
        sum[i] = sum[i-1] + a[i];
    }

    // 初始化: dp[i][i] = 0, 其余为 INF
    memset(dp, 0x3f, sizeof(dp));
    for (int i = 1; i <= n; i++)
        dp[i][i] = 0;

    // 区间DP: 按长度枚举
    for (int len = 2; len <= n; len++) {
        for (int l = 1; l + len - 1 <= n; l++) {
            int r = l + len - 1;
            for (int k = l; k < r; k++) {
                dp[l][r] = min(dp[l][r],
                    dp[l][k] + dp[k+1][r] + sum[r] - sum[l-1]);
            }
        }
    }

    printf("%d\n", dp[1][n]);
    return 0;
}

前缀和优化说明

如果不使用前缀和，每次计算 cost(l, k, r) 需要 O(r-l+1) 的时间遍历求和，总复杂度会退化为 O(n4)。使用前缀和后：

// 区间 [l, r] 的和可以在 O(1) 内求出
int range_sum = sum[r] - sum[l-1];

这样总时间复杂度保持在 O(n3)。

注意：环形石子合并

如果石子排成 环形，需要使用 "断环成链" 技巧（见第 6 节），将数组复制一倍后在长度为 2n 的序列上做区间 DP，答案为 max(dp[i][i+n-1])（i 从 1 到 n）。

经典题目

洛谷 P1880 [NOI1995] 石子合并

环形石子合并，分别求最小值和最大值。使用断环成链技巧。

3. 矩阵链乘法 (Matrix Chain Multiplication)

矩阵链乘法问题

给定 n 个矩阵 A1, A2, ..., An，其中 Ai 的维度为 pi-1 x pi。矩阵乘法满足结合律，不同的加括号方式会导致不同的乘法次数。求使总乘法次数最少的加括号方案。

例如：A1(10x30), A2(30x5), A3(5x60)

  • (A1A2)A3: 10*30*5 + 10*5*60 = 1500 + 3000 = 4500
  • A1(A2A3): 30*5*60 + 10*30*60 = 9000 + 18000 = 27000

状态定义

状态

dp[i][j] = 计算矩阵链 Ai x Ai+1 x ... x Aj 所需的最少乘法次数。

维度数组 p[0..n]，其中 Ai 的维度为 p[i-1] x p[i]。

转移方程

// 在 k 处分割: (A_i ... A_k) x (A_{k+1} ... A_j)
dp[i][j] = min( dp[i][k] + dp[k+1][j] + p[i-1] * p[k] * p[j] )   // i <= k < j

// 边界: dp[i][i] = 0 （单个矩阵不需要乘法）
// 答案: dp[1][n]

DP 表填充过程

以 p = [10, 30, 5, 60] 为例（3 个矩阵）：

  Step 1: 长度=1, dp[i][i] = 0

         A1    A2    A3
  A1     [0]
  A2           [0]
  A3                 [0]

  Step 2: 长度=2
    dp[1][2] = p0*p1*p2 = 10*30*5  = 1500
    dp[2][3] = p1*p2*p3 = 30*5*60  = 9000

         A1    A2    A3
  A1     [0]  1500
  A2           [0]  9000
  A3                 [0]

  Step 3: 长度=3
    dp[1][3]:
      k=1: dp[1][1] + dp[2][3] + p0*p1*p3 = 0 + 9000 + 10*30*60  = 27000
      k=2: dp[1][2] + dp[3][3] + p0*p2*p3 = 1500 + 0 + 10*5*60  = 4500
      min = 4500

  答案: dp[1][3] = 4500 (最优方案: (A1*A2)*A3)

完整 C++ 代码

#include <bits/stdc++.h>
using namespace std;

const int N = 110;
const int INF = 0x3f3f3f3f;

int n;
int p[N];         // 维度数组, A_i 为 p[i-1] x p[i]
int dp[N][N];      // dp[i][j] = 最少乘法次数
int split[N][N];   // 记录最优分割点 (用于构造方案)

int main() {
    scanf("%d", &n);
    for (int i = 0; i <= n; i++)
        scanf("%d", &p[i]);

    memset(dp, 0x3f, sizeof(dp));
    for (int i = 1; i <= n; i++)
        dp[i][i] = 0;

    for (int len = 2; len <= n; len++) {
        for (int i = 1; i + len - 1 <= n; i++) {
            int j = i + len - 1;
            for (int k = i; k < j; k++) {
                int cost = dp[i][k] + dp[k+1][j] + p[i-1] * p[k] * p[j];
                if (cost < dp[i][j]) {
                    dp[i][j] = cost;
                    split[i][j] = k;
                }
            }
        }
    }

    printf("%d\n", dp[1][n]);
    return 0;
}

构造最优加括号方案

// 递归打印最优括号方案
void print(int i, int j) {
    if (i == j) {
        printf("A%d", i);
        return;
    }
    printf("(");
    print(i, split[i][j]);
    print(split[i][j] + 1, j);
    printf(")");
}
与石子合并的区别

矩阵链乘法的 cost 不再是简单的区间和，而是两个子链矩阵相乘的代价 p[i-1] * p[k] * p[j]，这是区分两个模型的关键。但整体框架完全一致。

4. 括号匹配问题

UVa 1626 - Brackets Sequence

给定一个由 (, ), [, ] 组成的括号序列，求最少需要添加多少个括号才能使其成为一个合法的括号序列，并输出结果。

状态定义

状态

dp[l][r] = 使区间 [l, r] 变为合法括号序列需要添加的最少括号数。

转移方程

// 情况1: s[l] 和 s[r] 匹配，可以直接配对
if (s[l] 匹配 s[r])
    dp[l][r] = min(dp[l][r], dp[l+1][r-1]);

// 情况2: 在 k 处分割为两个子问题
dp[l][r] = min(dp[l][r], dp[l][k] + dp[k+1][r]);  // l <= k < r

// 边界: dp[i][i] = 1 (单个字符至少需要配一个)
//       dp[i][i-1] = 0 (空串合法, 处理长度为偶数时的边界)

括号匹配条件

左括号匹配的右括号
()
[]

DP 表填充过程

以 s = "([])" 为例：

  字符:   (   [   ]   )
  下标:   1   2   3   4

  Step 1: 长度=1, dp[i][i] = 1
       1   2   3   4
  1   [1]
  2        [1]
  3             [1]
  4                  [1]

  Step 2: 长度=2
    dp[1][2]: '(' 和 '[' 不匹配 -> min(dp[1][1]+dp[2][2]) = 2
    dp[2][3]: '[' 和 ']' 匹配   -> dp[3][2] = 0 (空串)   = 0
    dp[3][4]: ']' 和 ')' 不匹配 -> min(dp[3][3]+dp[4][4]) = 2

       1   2   3   4
  1   [1]  2
  2        [1]  0
  3             [1]  2
  4                  [1]

  Step 3: 长度=3
    dp[1][3]: '(' 和 ']' 不匹配
              k=1: dp[1][1]+dp[2][3] = 1+0 = 1
              k=2: dp[1][2]+dp[3][3] = 2+1 = 3
              min = 1
    dp[2][4]: '[' 和 ')' 不匹配
              k=2: dp[2][2]+dp[3][4] = 1+2 = 3
              k=3: dp[2][3]+dp[4][4] = 0+1 = 1
              min = 1

  Step 4: 长度=4
    dp[1][4]: '(' 和 ')' 匹配 -> dp[2][3] = 0
              k=1: dp[1][1]+dp[2][4] = 1+1 = 2
              k=2: dp[1][2]+dp[3][4] = 2+2 = 4
              k=3: dp[1][3]+dp[4][4] = 1+1 = 2
              min = 0

  答案: dp[1][4] = 0 (原串已经合法)

完整 C++ 代码

#include <bits/stdc++.h>
using namespace std;

const int N = 110;
const int INF = 0x3f3f3f3f;

int dp[N][N];
char s[N];
int n;

bool match(char a, char b) {
    return (a == '(' && b == ')') || (a == '[' && b == ']');
}

int main() {
    scanf("%s", s + 1);
    n = strlen(s + 1);

    memset(dp, 0x3f, sizeof(dp));
    for (int i = 1; i <= n; i++) {
        dp[i][i] = 1;               // 单字符需要一个配对
        dp[i][i-1] = 0;            // 空串合法
    }

    for (int len = 2; len <= n; len++) {
        for (int l = 1; l + len - 1 <= n; l++) {
            int r = l + len - 1;
            // 情况1: s[l] 和 s[r] 配对
            if (match(s[l], s[r]))
                dp[l][r] = min(dp[l][r], dp[l+1][r-1]);
            // 情况2: 枚举分割点
            for (int k = l; k < r; k++)
                dp[l][r] = min(dp[l][r], dp[l][k] + dp[k+1][r]);
        }
    }

    printf("%d\n", dp[1][n]);
    return 0;
}

输出构造方案的代码

// 递归输出添加最少括号后的合法序列
void print(int l, int r) {
    if (l > r) return;
    if (l == r) {
        if (s[l] == '(' || s[l] == ')') printf("()");
        else printf("[]");
        return;
    }
    // 情况1: s[l] 和 s[r] 匹配
    if (match(s[l], s[r]) && dp[l][r] == dp[l+1][r-1]) {
        putchar(s[l]);
        print(l+1, r-1);
        putchar(s[r]);
        return;
    }
    // 情况2: 枚举分割点
    for (int k = l; k < r; k++) {
        if (dp[l][r] == dp[l][k] + dp[k+1][r]) {
            print(l, k);
            print(k+1, r);
            return;
        }
    }
}

5. 戳气球 (Burst Balloons) - 逆向思维

LeetCode 312 - Burst Balloons

有 n 个气球，每个气球上有一个数字。戳破第 i 个气球可以获得 nums[left] * nums[i] * nums[right] 个金币，其中 left 和 right 是第 i 个气球左右两侧 尚未被戳破 的气球。求戳破所有气球能获得的最大金币数。

注意：边界外的气球数字视为 1。nums = [3, 1, 5, 8]，答案为 167。

为什么需要逆向思维

如果正向思考 "先戳哪个"，戳破一个气球后，左右邻居会改变，状态难以定义。

逆向思维：假设我们是 "添加气球" 而不是 "戳破气球"。考虑 最后被戳破 的气球 k，此时区间 [l, r] 内只有气球 k 还存活，它的左右邻居就是 l 和 r 边界上的值。

关键洞察 (逆向思维)

设 dp[l][r] = 戳破开区间 (l, r) 内所有气球能获得的最大金币数（不包括 l 和 r）。

假设 k 是区间 (l, r) 内 最后一个 被戳破的气球，则：

dp[l][r] = max( dp[l][k] + dp[k][r] + nums[l] * nums[k] * nums[r] )

戳破 k 时，(l, r) 内的其他气球都已被戳破，所以 k 的左右邻居就是 l 和 r。

状态定义与转移

// 在原数组首尾各加一个 1: val[0] = val[n+1] = 1
// dp[l][r] = 戳破 (l, r) 内所有气球的最大收益, l < r

dp[l][r] = max( dp[l][k] + dp[k][r] + val[l]*val[k]*val[r] )   // l < k < r

// 边界: dp[l][r] = 0, 当 r - l <= 1 (区间内没有气球)
// 答案: dp[0][n+1]

DP 表填充过程

以 nums = [3, 1, 5]，扩展为 val = [1, 3, 1, 5, 1]：

  val:  1   3   1   5   1
  idx:  0   1   2   3   4

  长度=2 (间隔1): dp[l][r] = 0 (区间内无气球)

  长度=3 (间隔2):
    dp[0][2] = val[0]*val[1]*val[2] = 1*3*1 = 3  (最后戳1)
    dp[1][3] = val[1]*val[2]*val[3] = 3*1*5 = 15 (最后戳2)
    dp[2][4] = val[2]*val[3]*val[4] = 1*5*1 = 5  (最后戳3)

  长度=4 (间隔3):
    dp[0][3]:
      k=1: dp[0][1]+dp[1][3]+val[0]*val[1]*val[3] = 0+15+1*3*5 = 30
      k=2: dp[0][2]+dp[2][3]+val[0]*val[2]*val[3] = 3+0+1*1*5  = 8
      max = 30
    dp[1][4]:
      k=2: dp[1][2]+dp[2][4]+val[1]*val[2]*val[4] = 0+5+3*1*1  = 8
      k=3: dp[1][3]+dp[3][4]+val[1]*val[3]*val[4] = 15+0+3*5*1 = 30
      max = 30

  长度=5 (间隔4):
    dp[0][4]:
      k=1: dp[0][1]+dp[1][4]+val[0]*val[1]*val[4] = 0+30+1*3*1  = 33
      k=2: dp[0][2]+dp[2][4]+val[0]*val[2]*val[4] = 3+5+1*1*1   = 9
      k=3: dp[0][3]+dp[3][4]+val[0]*val[3]*val[4] = 30+0+1*5*1  = 35
      max = 35

  答案: dp[0][4] = 35

完整 C++ 代码

#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    int maxCoins(vector<int>& nums) {
        int n = nums.size();
        // 扩展数组: 首尾加1
        vector<int> val(n + 2);
        val[0] = val[n + 1] = 1;
        for (int i = 1; i <= n; i++)
            val[i] = nums[i - 1];

        // dp[l][r] = 戳破 (l,r) 内所有气球的最大收益
        vector<vector<int>> dp(n + 2, vector<int>(n + 2, 0));

        // 按区间长度枚举 (注意这里长度的含义是间隔)
        for (int len = 2; len <= n + 1; len++) {
            for (int l = 0; l + len <= n + 1; l++) {
                int r = l + len;
                for (int k = l + 1; k < r; k++) {
                    dp[l][r] = max(dp[l][r],
                        dp[l][k] + dp[k][r] + val[l] * val[k] * val[r]);
                }
            }
        }

        return dp[0][n + 1];
    }
};
重要区别

戳气球问题的状态定义是 开区间 (l, r)，即 l 和 r 不被包含在区间内。这是因为逆向思维中，最后戳破的气球 k 的 "邻居" 是 l 和 r，它们必须在区间外面。这与石子合并的 闭区间 [l, r] 不同。

6. 环形区间 DP - 断环成链

问题背景

当石子排成 环形 时，第 1 堆和第 n 堆相邻。区间 DP 通常处理的是线性序列，如何将环形问题转化为线性问题？

断环成链技巧

核心思想

将长度为 n 的环形数组 复制一倍，得到长度为 2n 的线性数组。然后在新数组上做标准的区间 DP。

环形上任意一段连续的 n 个元素，都对应新数组上某个长度为 n 的区间。

  原始环形数组 (n=4):

       a1 ---- a2
      /          \
    a4           a3
      \          /
       ---------

  断环成链后 (长度 2n=8):

    a1  a2  a3  a4  a1  a2  a3  a4
    |_______________|
    从a1开始的一段

        |_______________|
        从a2开始的一段

            |_______________|
            从a3开始的一段

                |_______________|
                从a4开始的一段

  答案 = max/min( dp[i][i+n-1] ),  1 <= i <= n

实现步骤

将数组 a[1..n] 复制为 a[n+1..2n]，即 a[i+n] = a[i]
对长度为 2n 的数组计算前缀和
在新数组上做标准区间 DP，区间范围 [1, 2n]
答案为 min(dp[i][i+n-1])，其中 1 <= i <= n

完整 C++ 代码 - 环形石子合并

#include <bits/stdc++.h>
using namespace std;

const int N = 410;       // 注意: 开两倍空间
const int INF = 0x3f3f3f3f;

int n;
int a[N], sum[N];
int dp_min[N][N], dp_max[N][N];

int main() {
    scanf("%d", &n);

    // 读入并断环成链
    for (int i = 1; i <= n; i++) {
        scanf("%d", &a[i]);
        a[i + n] = a[i];          // 复制一倍
    }

    // 前缀和 (长度 2n)
    for (int i = 1; i <= 2 * n; i++)
        sum[i] = sum[i - 1] + a[i];

    // 初始化
    memset(dp_min, 0x3f, sizeof(dp_min));
    memset(dp_max, 0, sizeof(dp_max));
    for (int i = 1; i <= 2 * n; i++)
        dp_min[i][i] = 0;

    // 区间DP
    for (int len = 2; len <= n; len++) {
        for (int l = 1; l + len - 1 <= 2 * n; l++) {
            int r = l + len - 1;
            for (int k = l; k < r; k++) {
                int cost = sum[r] - sum[l - 1];
                dp_min[l][r] = min(dp_min[l][r],
                    dp_min[l][k] + dp_min[k+1][r] + cost);
                dp_max[l][r] = max(dp_max[l][r],
                    dp_max[l][k] + dp_max[k+1][r] + cost);
            }
        }
    }

    // 枚举所有起点, 取最优
    int ans_min = INF, ans_max = 0;
    for (int i = 1; i <= n; i++) {
        ans_min = min(ans_min, dp_min[i][i + n - 1]);
        ans_max = max(ans_max, dp_max[i][i + n - 1]);
    }

    printf("%d\n%d\n", ans_min, ans_max);
    return 0;
}

其他环形区间 DP 问题

断环成链技巧适用于所有环形区间 DP 问题，常见的有：

  • 环形石子合并：求最大/最小合并代价
  • 环形取珠子：环形序列上取相邻元素的最大收益
  • 多边形三角剖分：环形多边形的最优三角剖分
空间注意

断环成链后数组长度变为 2n，DP 表大小为 O((2n)2) = O(4n2)。注意开足够的空间，避免数组越界。

7. 总结与对比

区间 DP 通用模板

// 区间 DP 通用框架
#include <bits/stdc++.h>
using namespace std;

const int N = 510;
const int INF = 0x3f3f3f3f;

int n;
int dp[N][N];      // 状态数组
// 可能需要的辅助数组: 前缀和、维度数组等

int solve() {
    // 1. 初始化: dp[i][i] = 边界值
    memset(dp, 0x3f, sizeof(dp));
    for (int i = 1; i <= n; i++)
        dp[i][i] = 0;

    // 2. 按区间长度从小到大枚举
    for (int len = 2; len <= n; len++) {
        for (int l = 1; l + len - 1 <= n; l++) {
            int r = l + len - 1;
            // 3. 枚举分割点, 更新 dp[l][r]
            for (int k = l; k < r; k++) {
                dp[l][r] = min(dp[l][r],    // 或 max
                    dp[l][k] + dp[k+1][r] + cost(l, k, r));
            }
        }
    }

    return dp[1][n];
}

各模型对比

模型 状态定义 cost 函数 求解目标 特殊技巧
石子合并 dp[l][r] 闭区间 sum[r] - sum[l-1] min 代价 前缀和优化
矩阵链乘 dp[i][j] 闭区间 p[i-1]*p[k]*p[j] min 乘法次数 记录分割点
括号匹配 dp[l][r] 闭区间 +1 (不匹配时) min 添加数 两端匹配特判
戳气球 dp[l][r] 开区间 val[l]*val[k]*val[r] max 金币 逆向思维
环形区间DP dp[l][r] 闭区间 因题而异 因题而异 断环成链

常见优化

四边形不等式优化

如果 cost 函数满足 四边形不等式（quadrangle inequality），即：

w(a, c) + w(b, d) <= w(a, d) + w(b, c)，其中 a <= b <= c <= d

则分割点具有单调性，可以将枚举分割点的范围缩小，将时间复杂度从 O(n3) 优化到 O(n2)。

石子合并问题就满足四边形不等式条件。

解题方法论

确定状态：明确 dp[l][r] 代表什么含义（闭区间还是开区间）
写转移方程：枚举分割点 k，写出 dp[l][r] 与子区间状态的关系
确定边界：长度为 1 的区间（或长度为 0 的开区间）的初始值
确定填充顺序：按区间长度从小到大枚举
处理特殊情况：环形问题使用断环成链
输出答案：dp[1][n] 或 dp[0][n+1] 等
一句话总结

区间 DP = 枚举区间长度 + 枚举左右端点 + 枚举分割点。框架固定，关键在于正确识别 cost 函数和状态的开闭区间。

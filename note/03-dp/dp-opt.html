1. DP 优化概述

为什么要优化 DP？

动态规划的核心思想是用空间换时间，通过记忆化子问题的解避免重复计算。但很多时候，朴素 DP 的时间复杂度仍然很高（如 O(n³) 或 O(n²m)），在 n 达到 10∼10 量级时无法通过。此时需要利用问题的特殊结构来优化转移过程。

常见优化思路

优化技术 适用条件 典型复杂度优化
单调队列优化 转移方程中有滑动窗口上的最值查询 O(n²) → O(n)
斜率优化 (CHT) 转移方程可化为 y = kx + b 的形式 O(n²) → O(n) 或 O(n log n)
四边形不等式优化 区间 DP 中代价函数满足四边形不等式 O(n³) → O(n²)
矩阵快速幂优化 线性递推关系、转移矩阵可快速幂 O(n) → O(log n)
分治优化 (CDQ) 转移下标满足单调性，可分治处理 O(n²) → O(n log n)
决策单调性优化 最优决策点随状态单调移动 O(n²) → O(n log n)
优化的核心思想

DP 优化的本质是减少无效的决策枚举。通过证明决策的单调性、凸性等性质，我们可以用更高效的数据结构或算法跳过大量不可能成为最优解的决策点。关键步骤是：观察转移方程 → 提取数学结构 → 利用性质剪枝。

2. 单调队列优化 DP

适用场景

当 DP 转移方程中，需要在某个滑动窗口 [i-w, i-1] 内求最大值或最小值时，可以使用单调队列将转移从 O(w) 优化到 O(1) 均摊。

模式识别

转移方程形如：dp[i] = max/min{ dp[j] + f(j) }，其中 i - w ≤ j ≤ i - 1

或：dp[i] = max/min{ dp[j] + g(i, j) }，其中 g(i, j) 可拆分为只含 i 和只含 j 的部分

经典问题 1：滑动窗口最大值

给定长度为 n 的数组和窗口大小 k，求每个长度为 k 的连续子数组的最大值。这是理解单调队列的基础。

算法原理

维护一个单调递减的双端队列 (deque)，队列中存放的是数组下标
遍历数组，对于每个位置 i：先弹出队首过期元素（下标 < i - k + 1）
从队尾弹出所有值小于 a[i] 的元素（维护单调性），然后将 i 入队
队首即为当前窗口的最大值

#include <bits/stdc++.h>
using namespace std;

// 滑动窗口最大值 - 单调队列模板
vector<int> slidingWindowMax(vector<int>& a, int k) {
    deque<int> dq;          // 存下标，队首到队尾对应值单调递减
    vector<int> res;
    for (int i = 0; i < a.size(); i++) {
        // 1. 弹出过期元素（超出窗口左边界）
        while (!dq.empty() && dq.front() < i - k + 1)
            dq.pop_front();
        // 2. 维护单调递减：弹出队尾所有 < a[i] 的元素
        while (!dq.empty() && a[dq.back()] < a[i])
            dq.pop_back();
        // 3. 当前元素入队
        dq.push_back(i);
        // 4. 队首即为窗口最大值
        if (i >= k - 1)
            res.push_back(a[dq.front()]);
    }
    return res;
}

给定 n 个数和窗口大小 k，分别求每个窗口的最大值和最小值。

经典问题 3：切木条 (Cut Ribbon 的变体)

Luogu P2627 - 修剪草坪

有 n 头奶牛，第 i 头效率为 E[i]。选择若干头奶牛工作，但不能有连续超过 k 头奶牛同时工作。求最大效率总和。

DP 转移方程推导

设 dp[i] 表示考虑前 i 头奶牛，且第 i 头必须不工作时的最大效率。

dp[i] = max{ dp[j] + sum(j+1, i-1) }，其中 max(0, i-k-1) ≤ j ≤ i-1

展开得 dp[i] = max{ dp[j] + S[i-1] - S[j] }，其中 S 是前缀和。

整理：dp[i] = S[i-1] + max{ dp[j] - S[j] }，j 在滑动窗口内。

于是问题转化为在滑动窗口中求 dp[j] - S[j] 的最大值 — 典型的单调队列优化！

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

// Luogu P2627 修剪草坪 - 单调队列优化DP
int main() {
    int n, k;
    scanf("%d%d", &n, &k);

    vector<ll> E(n + 1), S(n + 1, 0);
    for (int i = 1; i <= n; i++) {
        scanf("%lld", &E[i]);
        S[i] = S[i - 1] + E[i];
    }

    // dp[i]: 前 i 头中，第 i 头不工作时的最大效率
    vector<ll> dp(n + 1, 0);
    deque<int> dq;
    dq.push_back(0);  // 哨兵

    for (int i = 1; i <= n; i++) {
        // 弹出过期决策: j < i - k - 1
        while (!dq.empty() && dq.front() < i - k - 1)
            dq.pop_front();

        // dp[i] = S[i-1] + max{ dp[j] - S[j] }
        dp[i] = S[i - 1] + (dp[dq.front()] - S[dq.front()]);

        // 维护单调递减：以 val = dp[i] - S[i] 的值为基准
        while (!dq.empty() &&
               dp[dq.back()] - S[dq.back()] <= dp[i] - S[i])
            dq.pop_back();
        dq.push_back(i);
    }

    // 答案: 要么全部工作(最后一个也工作), 要么dp[n](最后一个不工作)
    printf("%lld\n", max(dp[n], dp[n - 1] + E[n]));
    return 0;
}

单调队列优化 DP 通用模板

// 通用框架: dp[i] = min/max{ dp[j] + cost(j) }，j 在 [i-w, i-1] 内
deque<int> dq;
for (int i = 1; i <= n; i++) {
    // Step 1: 加入新决策 i-1 (扩大窗口右端)
    while (!dq.empty() && val(dq.back()) <= val(i - 1)) // 求max用 <=，求min用 >=
        dq.pop_back();
    dq.push_back(i - 1);

    // Step 2: 弹出过期决策 (窗口左端收缩)
    while (!dq.empty() && dq.front() < i - w)
        dq.pop_front();

    // Step 3: 取队首作为最优决策
    dp[i] = compute(dq.front());
}

3. 斜率优化 DP (Convex Hull Trick)

适用场景

当 DP 转移方程经过代数变形后，可以写成一次函数 y = kx + b 的形式时，可以用维护凸包的方式在 O(n log n) 甚至 O(n) 内完成转移。

模式识别

转移方程中含有 j²、j*S[j] 等二次项，且可以将 dp[j] 和 dp[i] 分离到等式两边。通常需要令 y = dp[j] + ...，x = f(j)，k = g(i)，b = dp[i]。

数学推导

以经典的任务安排问题为例：

有 n 个任务，每个任务有处理时间 t[i] 和费用 c[i]。将任务分成若干批，第 i 批的代价为 S × sum_c(第i批之后的所有任务)。求最小总费用。

朴素转移方程

dp[i] = min{ dp[j] + S * (C[n] - C[j]) + T[i] * (C[i] - C[j]) }

其中 T[i], C[i] 分别是 t 和 c 的前缀和。

变形过程

展开：dp[i] = min{ dp[j] + S*C[n] - S*C[j] + T[i]*C[i] - T[i]*C[j] }
将与 j 无关的项提出：dp[i] = S*C[n] + T[i]*C[i] + min{ dp[j] - S*C[j] - T[i]*C[j] }
整理 min 内：dp[j] = (S + T[i]) * C[j] + (dp[i] - S*C[n] - T[i]*C[i])
视为直线：y = kx + b，其中 y = dp[j]，x = C[j]，k = S + T[i]，b = dp[i] - ...
对于固定的 i，k 确定。要使 dp[i] 最小，即要使截距 b 最小，等价于找凸包上的最优切点
  y (dp[j])
  ^
  |     *  *
  |   *      \*       凸包上半部分（求最小值用下凸壳）
  |  *        |\
  | *    k    | \
  |*----->    v  *
  |            *
  +-------------------> x (C[j])

  斜率 k 随 i 递增（因为 T[i] 单调递增），
  所以最优决策点在凸包上单调右移

两种情况对比

特性 斜率单调（常见） 斜率不单调
凸包维护方式 队列（只在队尾插入，队首删除） 平衡树 / 二分
查询方式 队首弹出直到最优 在凸包上二分查找
总复杂度 TimeO(n) TimeO(n log n)

斜率计算与判断

精度陷阱

不要用浮点数计算斜率！用叉积判断三点是否构成凸壳：
对于三个点 A(x1,y1), B(x2,y2), C(x3,y3)，判断 B 是否应在凸包上：
(x2-x1)*(y3-y1) - (y2-y1)*(x3-x1) > 0 表示 A→B→C 逆时针（上凸壳保留 B）
求下凸壳则反之。实际使用时需要根据求最大/最小值以及斜率正负性仔细判断。

完整代码模板：斜率优化 DP（斜率单调）

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

/*
 * 斜率优化 DP 模板 - 以任务安排为例
 * dp[i] = min{ dp[j] + S*(C[n]-C[j]) + T[i]*(C[i]-C[j]) }
 * 变形: dp[j] = (S + T[i]) * C[j] + dp[i] - S*C[n] - T[i]*C[i]
 *       y    =         k        * x    +         b
 * 求最小截距 b，维护下凸壳
 */
int main() {
    int n;
    ll S;
    scanf("%d%lld", &n, &S);

    vector<ll> T(n + 1, 0), C(n + 1, 0);
    for (int i = 1; i <= n; i++) {
        ll t, c;
        scanf("%lld%lld", &t, &c);
        T[i] = T[i - 1] + t;
        C[i] = C[i - 1] + c;
    }

    vector<ll> dp(n + 1, 0);
    deque<int> dq;  // 凸包上的点（存下标）
    dq.push_back(0);

    for (int i = 1; i <= n; i++) {
        ll k = S + T[i];  // 当前查询斜率

        // 求最小截距：斜率单调递增时，弹出队首斜率 < k 的边
        while (dq.size() >= 2) {
            int a = dq[0], b = dq[1];
            // 若 (dp[b]-dp[a]) / (C[b]-C[a]) <= k, 弹出 a
            // 用叉积避免浮点: (dp[b]-dp[a]) <= k * (C[b]-C[a])
            if (dp[b] - dp[a] <= k * (C[b] - C[a]))
                dq.pop_front();
            else break;
        }
        int j = dq.front();

        // 用最优决策转移
        dp[i] = dp[j] + S * (C[n] - C[j]) + T[i] * (C[i] - C[j]);

        // 将 i 加入凸包，维护下凸壳性质
        while (dq.size() >= 2) {
            int a = dq[dq.size() - 2];
            int b = dq.back();
            // 判断 (dp[i]-dp[b])/(C[i]-C[b]) <= (dp[b]-dp[a])/(C[b]-C[a])
            // 即 b 点不在凸壳上，需要弹出
            if ((dp[i] - dp[b]) * (C[b] - C[a]) <= (dp[b] - dp[a]) * (C[i] - C[b]))
                dq.pop_back();
            else break;
        }
        dq.push_back(i);
    }

    printf("%lld\n", dp[n]);
    return 0;
}

经典例题详解

Luogu P3628 / BZOJ 1010 - 玩具装箱

有 n 个玩具，第 i 个长度为 L[i]。将它们分成若干组装箱，每组代价为 (sum(L) + 组内玩具数 - 1 - C)²，求最小总代价。

解题思路

设前缀和 S[i] = sum(L[1..i]) + i（包含间隔），转移方程为：

dp[i] = min{ dp[j] + (S[i] - S[j] - 1 - C)² }

令 A = S[i] - 1 - C，展开平方：

dp[i] = min{ dp[j] + A² - 2*A*S[j] + S[j]² }

整理：dp[j] + S[j]² = 2*A * S[j] + (dp[i] - A²)

令 y = dp[j] + S[j]²，x = S[j]，k = 2*A = 2*(S[i]-1-C)，b = dp[i] - A²

由于 A 随 i 递增，斜率 k 单调递增，用单调队列维护凸包即可 O(n) 解决。

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

// Luogu P3628 玩具装箱 - 斜率优化DP
ll L[50010], S[50010], dp[50010];
int q[50010];  // 手写队列

// 计算斜率：判断 f[j1] 和 f[j2] 对应的斜率
// f[j] = dp[j] + S[j]^2, x[j] = S[j]
// slope(j1, j2) = (f[j2] - f[j1]) / (x[j2] - x[j1])
double slope(int j1, int j2) {
    return (double)(dp[j2] + S[j2] * S[j2] - dp[j1] - S[j1] * S[j1])
         / (double)(S[j2] - S[j1]);
}

int main() {
    int n;
    ll C;
    scanf("%d%lld", &n, &C);

    for (int i = 1; i <= n; i++) {
        scanf("%lld", &L[i]);
        S[i] = S[i - 1] + L[i] + 1;  // 含间隔的前缀和
    }

    int head = 0, tail = 0;
    q[tail++] = 0;

    for (int i = 1; i <= n; i++) {
        ll A = S[i] - 1 - C;       // 查询斜率 k = 2*A

        // 弹出队首直到最优
        while (head + 1 < tail &&
               slope(q[head], q[head + 1]) <= 2.0 * A)
            head++;

        int j = q[head];
        dp[i] = dp[j] + (S[i] - S[j] - 1 - C) * (S[i] - S[j] - 1 - C);

        // 维护凸包
        while (head + 1 < tail &&
               slope(q[tail - 2], q[tail - 1]) >= slope(q[tail - 1], i))
            tail--;
        q[tail++] = i;
    }

    printf("%lld\n", dp[n]);
    return 0;
}

通用斜率优化模板（支持斜率不单调）

// 当查询斜率不单调时，在凸包上二分查找最优决策
vector<int> hull;  // 凸包上的点（按下标）

// 查询：在凸包上二分找到斜率 <= k 的最右位置
int query(ll k) {
    int lo = 0, hi = hull.size() - 1;
    while (lo < hi) {
        int mid = (lo + hi) / 2;
        // 比较 hull[mid] 和 hull[mid+1] 之间的斜率与 k
        if (getSlope(hull[mid], hull[mid + 1]) <= k)
            lo = mid + 1;
        else
            hi = mid;
    }
    return hull[lo];
}

// 插入：维护凸包，与单调队列方式相同
void insert(int i) {
    while (hull.size() >= 2 &&
           bad(hull[hull.size() - 2], hull.back(), i))
        hull.pop_back();
    hull.push_back(i);
}

4. 四边形不等式优化

适用场景

四边形不等式优化主要用于区间 DP 问题。当代价函数 w(i,j) 满足四边形不等式时，可以证明最优决策点具有单调性，从而将 O(n³) 优化到 O(n²)。

模式识别

区间 DP 转移方程形如：dp[i][j] = min{ dp[i][k] + dp[k+1][j] + w(i,j) }，其中 i ≤ k < j

且代价函数 w(i,j) 满足四边形不等式。

四边形不等式定义

四边形不等式 (Quadrangle Inequality)

函数 w(i,j) 满足四边形不等式，当且仅当对任意 a ≤ b ≤ c ≤ d，有：

w(a, c) + w(b, d) ≤ w(a, d) + w(b, c)

直观理解：矩阵 w 中，对于任意一个 2×2 的子矩阵，"对角线之和 ≤ 反对角线之和"。

区间上的单调性

若 w(i,j) 满足四边形不等式且 w(i,j) ≤ w(i+1,j)（区间包含单调性），则以下结论成立：

关键定理

设 s[i][j] 为 dp[i][j] 的最优决策点，则有：

s[i][j-1] ≤ s[i][j] ≤ s[i+1][j]

即最优决策点对列 j 单调不减，对行 i 单调不增。这将搜索范围从 O(n) 缩小到 s[i+1][j] - s[i][j-1]。

常见的满足四边形不等式的函数

函数形式 是否满足 QI 典型问题
w(i,j) = sum(a[i..j]) 是 石子合并
w(i,j) = (S[j]-S[i-1])² 是 邮局问题
w(i,j) = j - i 是（平凡情况） —
一般凸函数 通常是 需要具体验证

四边形不等式的验证方法

定义验证：直接验证 w(a,c) + w(b,d) ≤ w(a,d) + w(b,c)
导数判别法：对于连续函数 w，若 ∂²w/(∂i∂j) ≤ 0（交叉偏导非正），则满足
差分判别法：对于离散函数，验证 w(i,j+1) - w(i,j) 对 i 单调不增即可

经典例题：石子合并

Luogu P1880 / NOI 1995 - 石子合并

有 n 堆石子围成一圈，每次可以合并相邻两堆，代价为两堆石子之和。求合并成一堆的最小和最大代价。

朴素区间 DP（不优化）

// 朴素 O(n^3) 区间DP
// dp[i][j]: 合并 [i,j] 内石子的最小代价
// dp[i][j] = min{ dp[i][k] + dp[k+1][j] + w(i,j) }，i ≤ k < j
// w(i,j) = S[j] - S[i-1]

for (int len = 2; len <= n; len++) {
    for (int i = 1; i + len - 1 <= n; i++) {
        int j = i + len - 1;
        dp[i][j] = INF;
        for (int k = i; k < j; k++) {  // O(n) 枚举决策点
            dp[i][j] = min(dp[i][j], dp[i][k] + dp[k + 1][j] + w(i, j));
        }
    }
}

四边形不等式优化版本

// 四边形不等式优化 O(n^2)
// 关键: s[i][j-1] <= s[i][j] <= s[i+1][j]
// 枚举 k 的范围从 [i, j) 缩小到 [s[i][j-1], s[i+1][j]]

for (int i = 1; i <= n; i++) {
    s[i][i] = i;  // 边界：长度为1时决策点就是自身
    dp[i][i] = 0;
}

for (int len = 2; len <= n; len++) {
    for (int i = 1; i + len - 1 <= n; i++) {
        int j = i + len - 1;
        dp[i][j] = INF;
        // 利用决策单调性缩小枚举范围
        int left = s[i][j - 1];
        int right = s[i + 1][j];
        for (int k = left; k <= right; k++) {
            ll val = dp[i][k] + dp[k + 1][j] + w(i, j);
            if (val < dp[i][j]) {
                dp[i][j] = val;
                s[i][j] = k;
            }
        }
    }
}

完整代码：石子合并（四边形不等式优化）

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
const int N = 2010;
const ll INF = 1e18;

ll a[N], S[N];       // a[i]: 第i堆石子数, S: 前缀和
ll dp_min[N][N];      // 最小代价
ll dp_max[N][N];      // 最大代价
int smin[N][N];       // 最小值对应的最优决策点
int smax[N][N];       // 最大值对应的最优决策点

// 代价函数 w(i,j) = S[j] - S[i-1]
inline ll w(int i, int j) {
    return S[j] - S[i - 1];
}

int main() {
    int n;
    scanf("%d", &n);

    // 环形转链形：复制一倍
    for (int i = 1; i <= n; i++) {
        scanf("%lld", &a[i]);
        a[i + n] = a[i];
    }
    n *= 2;
    for (int i = 1; i <= n; i++)
        S[i] = S[i - 1] + a[i];

    // 初始化边界
    for (int i = 1; i <= n; i++) {
        smin[i][i] = i;
        smax[i][i] = i;
        dp_min[i][i] = 0;
        dp_max[i][i] = 0;
    }

    // 区间DP + 四边形不等式优化
    for (int len = 2; len <= n; len++) {
        for (int i = 1; i + len - 1 <= n; i++) {
            int j = i + len - 1;

            // 求最小值
            dp_min[i][j] = INF;
            for (int k = smin[i][j - 1]; k <= smin[i + 1][j]; k++) {
                ll val = dp_min[i][k] + dp_min[k + 1][j] + w(i, j);
                if (val < dp_min[i][j]) {
                    dp_min[i][j] = val;
                    smin[i][j] = k;
                }
            }

            // 求最大值 (注意: 最大值的决策单调性与最小值相反)
            dp_max[i][j] = -INF;
            for (int k = smax[i][j - 1]; k <= smax[i + 1][j]; k++) {
                ll val = dp_max[i][k] + dp_max[k + 1][j] + w(i, j);
                if (val > dp_max[i][j]) {
                    dp_max[i][j] = val;
                    smax[i][j] = k;
                }
            }
        }
    }

    // 在所有长度为 n 的区间中取最优
    ll ans_min = INF, ans_max = -INF;
    n /= 2;
    for (int i = 1; i <= n; i++) {
        ans_min = min(ans_min, dp_min[i][i + n - 1]);
        ans_max = max(ans_max, dp_max[i][i + n - 1]);
    }

    printf("%lld\n%lld\n", ans_min, ans_max);
    return 0;
}

邮局问题

Luogu P4767 / IOI 2000 - 邮局

在一条直线上有 n 个村庄和 p 个邮局，求如何放置邮局使得所有村庄到最近邮局的总距离最小。

DP 转移

dp[i][j] = min{ dp[i-1][k] + w(k+1, j) }，其中 w(l, r) 表示在 [l, r] 中设一个邮局的最小距离和。

w(l, r) 取 [l, r] 中位数位置的邮局即可。可以证明 w(l, r) 满足四边形不等式。

复杂度从 O(p×n²) 优化到 O(p×n)。

复杂度对比

方法 时间复杂度 空间复杂度 说明
朴素区间 DP TimeO(n³) O(n²) 枚举所有 i,j,k
四边形不等式优化 TimeO(n²) O(n²) 利用决策单调性缩小 k 的范围
四边形不等式优化注意事项
  • 必须先证明 w(i,j) 满足四边形不等式，否则优化是错误的
  • 区间长度 len 必须从小到大枚举，保证 s[i][j-1] 和 s[i+1][j] 已经计算
  • 最大值和最小值的决策单调性方向可能不同，需要分别推导
  • 环形问题先倍增为链形，再做区间 DP

5. 总结与对比

三大优化技术对比

特性 单调队列优化 斜率优化 四边形不等式优化
适用 DP 类型 线性 DP 线性 DP 区间 DP
转移特征 滑动窗口最值 可化为 y=kx+b 代价函数满足 QI

解题思路总结

写出朴素 DP 转移方程 — 这是一切优化的基础
分析转移方程的数学结构 — 是否有滑动窗口？是否可化为线性函数？代价函数是否有特殊性质？
选择合适的优化技术 — 根据方程结构匹配对应的优化方法
验证性质 — 单调性、凸性、四边形不等式是否成立
实现并调试 — 注意边界条件、精度问题（斜率优化避免浮点）
进阶方向
  • 决策单调性 + 分治优化：当转移满足决策单调性但不满足四边形不等式时，可用分治在 O(n log n) 内求解
  • WQS 二分（Alien 技巧）：当 DP 有附加限制（如恰好选 k 个）时，通过二分惩罚系数转化为无限制 DP
  • 李超线段树：维护动态凸包，支持插入直线和查询最大值，适合斜率不单调且在线的场景
  • CDQ 分治优化 DP：将 DP 的转移下标分治处理，配合数据结构优化

一、单调队列概述

1.1 什么是单调队列

单调队列是一种特殊的双端队列（deque），队列中的元素始终保持单调递增或单调递减的顺序。它结合了单调栈的思想和队列的滑动窗口特性，可以在 O(1) 时间内获取当前窗口内的最大值或最小值。

1.2 核心思想

单调队列的核心操作与单调栈类似，但增加了队首过期清理的功能：当队首元素已经滑出窗口时，将其从队首移除。

  单调队列 vs 单调栈：

  单调栈：只在一端操作（栈顶），用于求"下一个更大/小元素"
  单调队列：两端都可以操作，用于求"滑动窗口内的最大/小值"

  单调队列的两个关键操作：
  1. 维护单调性：从队尾弹出破坏单调性的元素
  2. 清理过期元素：从队首弹出超出窗口范围的元素

  入队：从队尾操作（类似单调栈）
  出队：从队首操作（清理过期元素）
  查询：队首就是当前窗口的最值

1.3 单调队列的类型

类型队列顺序队首元素典型应用
单调递增队列队首 < 队尾当前窗口最小值滑动窗口最小值
单调递减队列队首 > 队尾当前窗口最大值滑动窗口最大值

1.4 时间复杂度

每个元素最多入队一次、出队一次，总时间复杂度为 O(n)。单次查询最值为 O(1)。

总时间复杂度 O(n)
单次查询 O(1)
空间复杂度 O(n)
与优先队列的对比

优先队列（堆）也可以求滑动窗口最值，但每次操作是 O(log n)，且删除过期元素需要懒惰删除技巧。单调队列在每个元素最多操作两次的保证下，总复杂度为 O(n)，优于优先队列的 O(n log n)。

二、滑动窗口最大/小值

2.1 问题定义

给定一个长度为 n 的数组和一个大小为 k 的滑动窗口，窗口从数组的最左端滑到最右端。每次窗口向右移动一个位置，求每次移动后窗口内的最大值（或最小值）。

  数组: [1, 3, -1, -3, 5, 3, 6, 7]，窗口大小 k=3

  窗口位置          窗口内容        最大值
  [1, 3, -1]        1, 3, -1         3
     [3, -1, -3]    3, -1, -3        3
        [-1, -3, 5] -1, -3, 5        5
           [-3, 5, 3] -3, 5, 3       5
              [5, 3, 6] 5, 3, 6      6
                 [3, 6, 7] 3, 6, 7   7

  最大值序列: [3, 3, 5, 5, 6, 7]

2.2 算法步骤

初始化空双端队列，存下标
遍历数组的每个元素 a[i]（i 从 0 到 n-1）
清理队首：如果队首元素的下标已经超出窗口范围（即 队首 < i - k + 1），从队首弹出
维护单调性：从队尾弹出所有 < a[i] 的元素（求最大值时用单调递减队列）
将 i 入队
当 i >= k - 1 时，队首就是当前窗口的最大值

2.3 图解过程

  a = [1, 3, -1, -3, 5, 3, 6, 7], k = 3
  求滑动窗口最大值（单调递减队列，存下标）

  i=0, a[0]=1: 队列为空, 入队。队列: [0(1)]
  窗口未满 (i < k-1=2)

  i=1, a[1]=3: 3 > a[队尾]=1, 弹出队尾。入队。队列: [1(3)]
  窗口未满

  i=2, a[2]=-1: -1 <= a[队尾]=3, 入队。队列: [1(3), 2(-1)]
  窗口已满, 最大值 = a[队首] = a[1] = 3

  i=3, a[3]=-3: 队首下标1 >= 3-3+1=1, 不过期。
  -3 <= a[队尾]=-1, 入队。队列: [1(3), 2(-1), 3(-3)]
  最大值 = a[1] = 3

  i=4, a[4]=5: 队首下标1 < 4-3+1=2, 过期, 弹出队首。
  队列: [2(-1), 3(-3)]
  5 > a[队尾]=-3, 弹出。5 > a[队尾]=-1, 弹出。入队。
  队列: [4(5)]
  最大值 = a[4] = 5

  i=5, a[5]=3: 3 <= a[队尾]=5, 入队。队列: [4(5), 5(3)]
  最大值 = a[4] = 5

  i=6, a[6]=6: 队首下标4 < 6-3+1=4, 不过期。
  6 > a[队尾]=3, 弹出。6 > a[队尾]=5, 弹出。入队。
  队列: [6(6)]
  最大值 = a[6] = 6

  i=7, a[7]=7: 队首下标6 < 7-3+1=5, 不过期。
  7 > a[队尾]=6, 弹出。入队。
  队列: [7(7)]
  最大值 = a[7] = 7

  结果: [3, 3, 5, 5, 6, 7]

2.4 C++ 模板：滑动窗口最大值

#include <iostream>
#include <deque>
#include <vector>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    vector<int> a(n);
    for (int i = 0; i < n; i++)
        cin >> a[i];

    // 单调递减队列，存下标，队首为最大值
    deque<int> dq;
    vector<int> ans;

    for (int i = 0; i < n; i++) {
        // 1. 清理过期元素：队首下标超出窗口左边界
        while (!dq.empty() && dq.front() < i - k + 1)
            dq.pop_front();

        // 2. 维护单调性：从队尾弹出比当前元素小的
        while (!dq.empty() && a[dq.back()] <= a[i])
            dq.pop_back();

        // 3. 当前元素入队
        dq.push_back(i);

        // 4. 窗口满时记录答案
        if (i >= k - 1)
            ans.push_back(a[dq.front()]);
    }

    for (int x : ans)
        cout << x << " ";
    return 0;
}

2.5 滑动窗口最小值

只需将单调递减队列改为单调递增队列，即改变维护单调性的比较方向：

// 单调递增队列，队首为最小值
deque<int> dq;
for (int i = 0; i < n; i++) {
    // 清理过期元素
    while (!dq.empty() && dq.front() < i - k + 1)
        dq.pop_front();

    // 维护单调递增：从队尾弹出比当前元素大的
    while (!dq.empty() && a[dq.back()] >= a[i])
        dq.pop_back();

    dq.push_back(i);

    if (i >= k - 1)
        cout << a[dq.front()] << " ";
}

2.6 同时求最大值和最小值

// 同时维护两个队列，一次遍历得到最大值和最小值
deque<int> dq_max, dq_min;
for (int i = 0; i < n; i++) {
    // 清理过期
    while (!dq_max.empty() && dq_max.front() < i - k + 1) dq_max.pop_front();
    while (!dq_min.empty() && dq_min.front() < i - k + 1) dq_min.pop_front();

    // 维护单调性
    while (!dq_max.empty() && a[dq_max.back()] <= a[i]) dq_max.pop_back();
    while (!dq_min.empty() && a[dq_min.back()] >= a[i]) dq_min.pop_back();

    dq_max.push_back(i);
    dq_min.push_back(i);

    if (i >= k - 1) {
        int mx = a[dq_max.front()];
        int mn = a[dq_min.front()];
        // 使用最大值和最小值做进一步处理
    }
}
实现技巧

1. 队列中存下标而非值，方便判断元素是否过期
2. 过期判断：dq.front() < i - k + 1 表示队首元素已不在当前窗口内
3. 可以用 std::deque 或手写数组模拟双端队列（常数更小）

三、DP 优化应用

3.1 问题模式

当 DP 的转移方程中涉及区间最值时，可以使用单调队列优化。典型模式：

dp[i] = min/max{ dp[j] + cost(j, i) }    其中 j 属于某个滑动窗口

3.2 经典问题：最大子段和（滑动窗口版本）

给定数组和窗口大小 k，求长度不超过 k 的连续子段的最大和。

  问题：dp[i] = S[i] - min{ S[j] }, 其中 i-k <= j <= i-1

  S[i] 是前缀和，我们需要在窗口 [i-k, i-1] 中找到最小的 S[j]

  用单调递增队列维护前缀和的最小值：
  - 队首始终是当前窗口内的最小前缀和
  - dp[i] = S[i] - S[队首]

  时间复杂度: O(n)
#include <iostream>
#include <deque>
#include <climits>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;

    // 前缀和
    long long S = 0;
    long long ans = LLONG_MIN;
    deque<int> dq;  // 存下标，单调递增队列

    // 初始：S[0] = 0 入队
    dq.push_back(0);
    long long S_arr[n + 1];
    S_arr[0] = 0;

    for (int i = 1; i <= n; i++) {
        int x; cin >> x;
        S += x;
        S_arr[i] = S;

        // 清理过期元素
        while (!dq.empty() && dq.front() < i - k)
            dq.pop_front();

        // 更新答案：S[i] - S[最小的j]
        if (!dq.empty())
            ans = max(ans, S - S_arr[dq.front()]);

        // 维护单调递增
        while (!dq.empty() && S_arr[dq.back()] >= S)
            dq.pop_back();
        dq.push_back(i);
    }
    cout << ans << endl;
    return 0;
}

3.3 经典问题：修剪草坪

洛谷 P2627 修剪草坪：n 头牛排成一排，每头牛有一个效率值。选择一些牛组成团队，但不能连续选择超过 k 头牛。求团队效率总和的最大值。

  状态转移方程：
  dp[i] = max(dp[i-1],  max{ dp[j-1] + sum(j, i) })  (i-k <= j <= i)

  其中 sum(j, i) = S[i] - S[j-1]

  变形：dp[i] = max(dp[i-1], S[i] + max{ dp[j-1] - S[j-1] })
                (i-k <= j <= i)

  用单调递减队列维护 dp[j-1] - S[j-1] 的最大值
#include <iostream>
#include <deque>
using namespace std;

const int MAXN = 100005;
long long a[MAXN], S[MAXN], dp[MAXN];

int main() {
    int n, k;
    cin >> n >> k;

    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        S[i] = S[i-1] + a[i];
    }

    deque<int> dq;
    dq.push_back(0);  // 哨兵

    for (int i = 1; i <= n; i++) {
        // 清理过期
        while (!dq.empty() && dq.front() < i - k - 1)
            dq.pop_front();

        // 转移：选或不选第 i 头牛
        dp[i] = max(dp[i-1], S[i] + (dq.empty() ? 0 : dp[dq.front()-1] - S[dq.front()]));

        // 维护单调递减：dp[j-1] - S[j-1] 从大到小
        while (!dq.empty() &&
               dp[dq.back()-1] - S[dq.back()] <= dp[i-1] - S[i])
            dq.pop_back();
        dq.push_back(i);
    }
    cout << dp[n] << endl;
    return 0;
}

3.4 单调队列优化 DP 的一般步骤

写出朴素 DP 方程，找到转移中涉及区间最值的部分
分离变量：将转移式中只与 j 有关的部分和只与 i 有关的部分分开
确定队列类型：求最大值用单调递减队列，求最小值用单调递增队列
确定窗口范围：即 j 的合法取值范围，用于判断过期
实现：维护单调性 + 清理过期 + 更新答案
常见错误

1. 过期条件写错：注意窗口是 [i-k, i] 还是 [i-k, i-1]，边界差 1
2. 入队时机：有些问题需要先更新答案再入队，有些需要先入队再更新
3. 队列为空的判断：转移时需要检查队列是否为空

1. 强连通分量概述

在有向图中，如果顶点 u 和 v 互相可达（即存在 u → v 和 v → u 的路径），则称 u 和 v 强连通。

定义

强连通分量 (SCC, Strongly Connected Component) 是有向图中的极大强连通子图。即：分量内任意两点互相可达，且不能再加入更多顶点使其仍保持强连通。

直观理解

  有向图:                 强连通分量:

    1 → 2 → 3            SCC1 = {1, 2, 3}  (1→2→3→1)
    ↑   ↓   ↓            SCC2 = {4, 5}     (4→5→4)
    └───┘   ↓            SCC3 = {6}
    4 → 5 → 6

  分量间关系:  SCC1 → SCC3, SCC2 → SCC1

缩点 (Condensation)

将每个 SCC 缩成一个点，得到的新图称为缩点图（Condensation Graph）。缩点图一定是 DAG（有向无环图）。

这个性质非常重要：它将复杂的有向图问题转化为 DAG 上的问题，可以利用拓扑排序和 DAG 上的 DP 来解决。

为什么缩点图一定是 DAG?

假设缩点图中存在环，则环上的所有 SCC 可以合并为一个更大的 SCC（因为 SCC 之间互相可达），与 SCC 的极大性矛盾。

SCC 算法对比

算法遍历次数实现难度特点
Tarjan1 次 DFS中等一次 DFS 同时求出所有 SCC，效率高
Kosaraju2 次 DFS简单思路直观，需建反向图

2. Tarjan 算法

Tarjan 算法由 Robert Tarjan 于 1972 年提出，通过一次 DFS 即可求出所有强连通分量。

核心概念

dfn[u] — 时间戳

表示顶点 u 被 DFS 访问的时间顺序（第几个被访问）。dfn 值是全局递增的。

low[u] — 能回溯到的最小 dfn

表示从 u 出发，经过 u 的子树中的节点，再通过一条反向边（非树边）所能到达的最小 dfn 值。

栈 — 存储当前搜索路径

用栈记录当前 DFS 路径上的节点。当发现某个节点是 SCC 的根时，将栈中该节点以上的所有节点弹出，构成一个 SCC。

判断 SCC 根的条件

dfn[u] == low[u] 时，u 是当前 SCC 的"根"（栈底元素）。此时栈中从 u 到栈顶的所有节点构成一个 SCC。

重要理解

low[u] 不是简单地取所有子节点 low[v] 的最小值。对于树边 u → v，low[u] = min(low[u], low[v])；但对于非树边（反向边、横叉边），low[u] = min(low[u], dfn[v])（注意用的是 dfn[v] 而非 low[v]，且要求 v 在栈中）。

算法步骤

对图进行 DFS，为每个节点分配 dfn 和初始 low（low[u] = dfn[u] = ++timer），并将 u 入栈，标记 in_stack[u] = true。
遍历 u 的每个邻居 v：
  • 若 v 未访问：递归 DFS(v)，回溯后 low[u] = min(low[u], low[v])。
  • 若 v 在栈中：low[u] = min(low[u], dfn[v])（找到了一条反向边）。
  • 若 v 已访问但不在栈中：v 属于已处理的 SCC，忽略。
当 dfn[u] == low[u] 时，u 是 SCC 根。不断弹栈直到弹出 u，弹出的节点构成一个 SCC。
  Tarjan 算法执行过程:

  图: 1→2→3→1, 3→4→5→4

  DFS(1): dfn[1]=low[1]=1, 入栈 [1]
    DFS(2): dfn[2]=low[2]=2, 入栈 [1,2]
      DFS(3): dfn[3]=low[3]=3, 入栈 [1,2,3]
        3→1: 1在栈中, low[3]=min(3,dfn[1])=1
        3→4: DFS(4)
          dfn[4]=low[4]=4, 入栈 [1,2,3,4]
          4→5: DFS(5)
            dfn[5]=low[5]=5, 入栈 [1,2,3,4,5]
            5→4: 4在栈中, low[5]=min(5,dfn[4])=4
          回溯: low[4]=min(4,low[5])=4
          dfn[4]==low[4]? YES → 弹栈直到4: {5,4} → SCC2
        回溯: low[3]=min(1,low[4])=1
      回溯: low[2]=min(2,low[3])=1
    回溯: low[1]=min(1,low[2])=1
    dfn[1]==low[1]? YES → 弹栈直到1: {3,2,1} → SCC1

Tarjan 算法模板

// Tarjan 求强连通分量
vector<vector<int>> adj;
vector<int> dfn, low, in_stack, scc_id;
stack<int> stk;
int timer, scc_cnt;

void tarjan(int u) {
    dfn[u] = low[u] = ++timer;
    stk.push(u);
    in_stack[u] = 1;

    for (int v : adj[u]) {
        if (!dfn[v]) {
            // 树边: v 未访问
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (in_stack[v]) {
            // 反向边: v 在栈中 (属于当前搜索路径)
            low[u] = min(low[u], dfn[v]);
        }
        // 若 v 已访问且不在栈中, 说明 v 属于已处理的 SCC, 忽略
    }

    if (dfn[u] == low[u]) {
        // u 是 SCC 的根, 弹出整个 SCC
        scc_cnt++;
        while (true) {
            int x = stk.top(); stk.pop();
            in_stack[x] = 0;
            scc_id[x] = scc_cnt;
            if (x == u) break;
        }
    }
}

// 初始化并求解
void solve_scc(int n) {
    dfn.assign(n + 1, 0);
    low.assign(n + 1, 0);
    in_stack.assign(n + 1, 0);
    scc_id.assign(n + 1, 0);
    timer = scc_cnt = 0;

    for (int i = 1; i <= n; i++)
        if (!dfn[i]) tarjan(i);
}

3. Kosaraju 算法

Kosaraju 算法通过两次 DFS 求出所有强连通分量，思路直观且易于理解。

核心思想

Kosaraju 算法的关键洞察：在原图的反向图上，从 DFS 退出时间最晚的节点开始搜索，能够精确地探索一个完整的 SCC。

算法步骤

第一次 DFS：对原图进行 DFS，记录每个节点的退出时间（后序遍历顺序）。退出时间越晚的节点排在越后面。
构建反向图：将原图的所有边反向。
第二次 DFS：按退出时间从大到小的顺序，在反向图上进行 DFS。每次 DFS 能访问到的所有节点构成一个 SCC。
  Kosaraju 算法过程:

  原图: 1→2, 2→3, 3→1, 3→4, 4→5, 5→4

  第一次 DFS (原图, 记录后序):
    DFS(1)→DFS(2)→DFS(3)→DFS(4)→DFS(5)
    后序: [5, 4, 3, 2, 1]

  反向图: 2→1, 3→2, 1→3, 4→3, 5→4, 4→5

  第二次 DFS (反向图, 按后序逆序):
    访问 1: DFS(1)→3→2→{1,3,2} → SCC1
    访问 2: 已访问
    访问 3: 已访问
    访问 4: DFS(4)→5→{4,5} → SCC2
    访问 5: 已访问

为什么 Kosaraju 是正确的?

关键在于反向图的性质。假设在原图中有一个 SCC = {A, B, C}，那么在反向图中它们仍然互相可达。

第一次 DFS 确保：如果 SCC X 中的节点能到达 SCC Y 中的节点（原图中），则 X 中节点的退出时间一定晚于 Y。在反向图中按逆序访问，恰好保证了每次从"最上游"的 SCC 开始搜索，不会跨越 SCC 边界。

Kosaraju 算法模板

// Kosaraju 求强连通分量
vector<vector<int>> adj, radj;  // 原图, 反向图
vector<int> order, scc_id, vis;
int scc_cnt;

// 第一次 DFS: 后序遍历
void dfs1(int u) {
    vis[u] = 1;
    for (int v : adj[u])
        if (!vis[v]) dfs1(v);
    order.push_back(u);  // 退出时加入
}

// 第二次 DFS: 在反向图上标记 SCC
void dfs2(int u, int id) {
    scc_id[u] = id;
    for (int v : radj[u])
        if (!scc_id[v]) dfs2(v, id);
}

void kosaraju(int n) {
    vis.assign(n + 1, 0);
    order.clear();

    // 第一次 DFS
    for (int i = 1; i <= n; i++)
        if (!vis[i]) dfs1(i);

    // 第二次 DFS (按后序逆序)
    scc_id.assign(n + 1, 0);
    scc_cnt = 0;
    reverse(order.begin(), order.end());
    for (int u : order)
        if (!scc_id[u])
            dfs2(u, ++scc_cnt);
}
Tarjan vs Kosaraju 选择建议
  • 竞赛中推荐 Tarjan：只需一次 DFS，无需建反向图，空间更优。
  • 学习理解推荐 Kosaraju：思路直观，容易理解正确性。
  • 两者时间复杂度相同，均为 O(V + E)。

4. 缩点 — 构建 DAG

缩点 (Condensation) 是将每个 SCC 视为一个超级节点，SCC 之间的有向边保留，从而将原图转化为 DAG 的过程。

缩点步骤

使用 Tarjan 或 Kosaraju 算法求出所有 SCC，为每个节点标记 scc_id[u]。
遍历原图中的每条边 (u, v)：若 scc_id[u] != scc_id[v]，则在缩点图中添加边 scc_id[u] → scc_id[v]。
可选：去重（若不需要重边）。
  缩点过程:

  原图:  1→2→3→1, 2→4, 4→5→6→4, 6→7

  Tarjan 求 SCC:
    SCC1 = {1,2,3}   scc_id = 1
    SCC2 = {4,5,6}   scc_id = 2
    SCC3 = {7}        scc_id = 3

  原边映射:
    1→2: 1==1 (忽略)
    2→3: 1==1 (忽略)
    3→1: 1==1 (忽略)
    2→4: 1≠2 → 缩点图边 1→2
    4→5: 2==2 (忽略)
    5→6: 2==2 (忽略)
    6→4: 2==2 (忽略)
    6→7: 2≠3 → 缩点图边 2→3

  缩点图 (DAG): 1 → 2 → 3

缩点模板

// 缩点: 从原图构建 DAG
// adj: 原图邻接表, scc_id: Tarjan 求出的 SCC 编号
// 返回: 缩点图邻接表 (0-indexed SCC 编号)
vector<vector<int>> condensation(int n, int scc_cnt,
    const vector<vector<int>>& adj,
    const vector<int>& scc_id) {

    vector<vector<int>> dag(scc_cnt + 1);
    set<pair<int,int>> seen;  // 去重

    for (int u = 1; u <= n; u++) {
        for (int v : adj[u]) {
            int su = scc_id[u], sv = scc_id[v];
            if (su != sv && seen.insert({su, sv}).second)
                dag[su].push_back(sv);
        }
    }
    return dag;
}
SCC 内的信息聚合

缩点时通常需要对 SCC 内的节点信息进行聚合，例如：SCC 内所有节点的权值之和、SCC 的大小、SCC 内是否包含特殊节点等。在标记 scc_id 的同时或之后进行统计即可。

5. 缩点后 DAG 上的 DP

缩点后得到 DAG，可以利用拓扑排序进行各种 DP 问题的求解。这是 SCC 问题中最常见的应用模式。

典型问题模型

给定有向图，每个点有权值 w[u]，求一条路径使得经过的节点权值之和最大。

解题思路

用 Tarjan 算法求 SCC，得到 scc_id[]。
对每个 SCC 内的权值求和：scc_w[id] += w[u]。
缩点构建 DAG，同时记录每个 SCC 的聚合权值。
在 DAG 上按拓扑序 DP：dp[v] = max(dp[v], dp[u] + scc_w[v])。
答案为 max(dp[i])。

缩点 + DAG DP 完整代码

// 缩点 + DAG 最长路 DP
// 问题: 求有向图中经过节点权值之和最大的路径
#include <bits/stdc++.h>
using namespace std;

const int N = 10010;

vector<int> adj[N];
int dfn[N], low[N], scc_id[N], in_stack[N];
int w[N], scc_w[N];  // 节点权值, SCC 聚合权值
stack<int> stk;
int timer, scc_cnt, n;

void tarjan(int u) {
    dfn[u] = low[u] = ++timer;
    stk.push(u);
    in_stack[u] = 1;

    for (int v : adj[u]) {
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (in_stack[v]) {
            low[u] = min(low[u], dfn[v]);
        }
    }

    if (dfn[u] == low[u]) {
        scc_cnt++;
        while (true) {
            int x = stk.top(); stk.pop();
            in_stack[x] = 0;
            scc_id[x] = scc_cnt;
            scc_w[scc_cnt] += w[x];  // 聚合权值
            if (x == u) break;
        }
    }
}

int main() {
    int m;
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++) scanf("%d", &w[i]);
    for (int i = 0; i < m; i++) {
        int u, v; scanf("%d%d", &u, &v);
        adj[u].push_back(v);
    }

    // 1. Tarjan 求 SCC
    for (int i = 1; i <= n; i++)
        if (!dfn[i]) tarjan(i);

    // 2. 缩点构建 DAG
    vector<vector<int>> dag(scc_cnt + 1);
    vector<int> indeg(scc_cnt + 1, 0);
    set<pair<int,int>> seen;
    for (int u = 1; u <= n; u++) {
        for (int v : adj[u]) {
            int su = scc_id[u], sv = scc_id[v];
            if (su != sv && seen.insert({su, sv}).second) {
                dag[su].push_back(sv);
                indeg[sv]++;
            }
        }
    }

    // 3. 拓扑排序 + DP
    queue<int> q;
    vector<int> dp(scc_cnt + 1, 0);
    for (int i = 1; i <= scc_cnt; i++) {
        dp[i] = scc_w[i];
        if (!indeg[i]) q.push(i);
    }

    int ans = 0;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        ans = max(ans, dp[u]);
        for (int v : dag[u]) {
            dp[v] = max(dp[v], dp[u] + scc_w[v]);
            if (--indeg[v] == 0) q.push(v);
        }
    }

    printf("%d\n", ans);
    return 0;
}
常见 DP 转移
  • 最长路：dp[v] = max(dp[v], dp[u] + val[v])
  • 最短路：dp[v] = min(dp[v], dp[u] + val[v])
  • 路径计数：dp[v] = (dp[v] + dp[u]) % MOD
  • 方案判定：dp[v] |= dp[u] (可达性)

6.1 Tarjan 链式前向星版

// Tarjan SCC — 链式前向星存储
#include <bits/stdc++.h>
using namespace std;

const int N = 10010, M = 100010;

struct Edge { int to, nxt; } e[M];
int head[N], cnt;
int dfn[N], low[N], scc_id[N], in_stack[N];
stack<int> stk;
int timer, scc_cnt, n, m;

void add(int u, int v) {
    e[++cnt] = {v, head[u]};
    head[u] = cnt;
}

void tarjan(int u) {
    dfn[u] = low[u] = ++timer;
    stk.push(u);
    in_stack[u] = 1;

    for (int i = head[u]; i; i = e[i].nxt) {
        int v = e[i].to;
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (in_stack[v]) {
            low[u] = min(low[u], dfn[v]);
        }
    }

    if (dfn[u] == low[u]) {
        scc_cnt++;
        while (true) {
            int x = stk.top(); stk.pop();
            in_stack[x] = 0;
            scc_id[x] = scc_cnt;
            if (x == u) break;
        }
    }
}

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < m; i++) {
        int u, v; scanf("%d%d", &u, &v);
        add(u, v);
    }

    for (int i = 1; i <= n; i++)
        if (!dfn[i]) tarjan(i);

    printf("SCC 数量: %d\n", scc_cnt);
    for (int i = 1; i <= n; i++)
        printf("节点 %d 属于 SCC %d\n", i, scc_id[i]);
    return 0;
}

6.2 Kosaraju 完整模板

// Kosaraju SCC — 完整模板
#include <bits/stdc++.h>
using namespace std;

const int N = 10010;

vector<int> adj[N], radj[N];
vector<int> order, scc_id, vis;
int scc_cnt, n, m;

void dfs1(int u) {
    vis[u] = 1;
    for (int v : adj[u])
        if (!vis[v]) dfs1(v);
    order.push_back(u);
}

void dfs2(int u, int id) {
    scc_id[u] = id;
    for (int v : radj[u])
        if (!scc_id[v]) dfs2(v, id);
}

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < m; i++) {
        int u, v; scanf("%d%d", &u, &v);
        adj[u].push_back(v);
        radj[v].push_back(u);  // 反向边
    }

    // 第一次 DFS
    vis.assign(n + 1, 0);
    for (int i = 1; i <= n; i++)
        if (!vis[i]) dfs1(i);

    // 第二次 DFS
    scc_id.assign(n + 1, 0);
    scc_cnt = 0;
    reverse(order.begin(), order.end());
    for (int u : order)
        if (!scc_id[u])
            dfs2(u, ++scc_cnt);

    printf("SCC 数量: %d\n", scc_cnt);
    return 0;
}

6.3 2-SAT 中的 SCC 应用

2-SAT 问题是 SCC 的经典应用之一。对于布尔变量 x_i，每个变量拆成两个节点（x_i 和 ¬x_i），根据约束建边，然后用 Tarjan 判断是否有解。

// 2-SAT 判定 (利用 SCC)
// 若 x_i 和 ¬x_i 在同一个 SCC 中, 则无解
// 否则按 SCC 拓扑序贪心取值即可
bool check_2sat(int n) {
    for (int i = 0; i < n; i++)
        if (scc_id[i] == scc_id[i + n])
            return false;  // x 和 ¬x 在同一 SCC, 矛盾
    return true;
}

关于 low 数组的实现细节

不同教材对 low[u] 的更新方式有微小差异：有的用 dfn[v]，有的用 low[v]。对于 Tarjan SCC 算法，两种写法结果一致（都能正确求出 SCC）。但在求割点和桥时，必须严格区分 dfn[v] 和 low[v]。

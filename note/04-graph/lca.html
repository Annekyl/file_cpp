LCA 概述

定义

对于一棵有根树，节点 u 和节点 v 的最近公共祖先 (Lowest Common Ancestor) 是指同时是 u 和 v 的祖先，且深度最大的那个节点。

            1           <-- 根节点
           / \
          2   3
         / \   \
        4   5   6
       / \
      7   8

  LCA(4, 5) = 2
  LCA(4, 6) = 1
  LCA(7, 8) = 4
  LCA(5, 6) = 1
  LCA(3, 6) = 3   (自身也是祖先)

应用场景

  • 树上两点距离：dist(u, v) = depth[u] + depth[v] - 2 * depth[LCA(u, v)]
  • 树上路径信息：求路径上的最大/最小边权、路径异或和等
  • 树上差分：对 u 到 v 路径上的所有节点加值
  • 树的直径：配合 DFS/BFS 求最远点对
  • 在线查询：多次询问树上两节点关系
LCA 算法分类
  • 在线算法：倍增法、ST 表 + DFS 序、树链剖分 — 预处理后每次查询 O(log n) 或 O(1)
  • 离线算法：Tarjan 算法 — 所有查询一起处理，总复杂度 O(n + q)

倍增法 (Binary Lifting)

核心思想

预处理每个节点向上跳 2^k 步到达的祖先，记为 fa[u][k]。查询时将两个节点同时向上跳，从大步长到小步长逐步逼近。

预处理

通过 DFS 求出每个节点的深度和直接父亲 fa[u][0]，然后用递推：

// fa[u][k] = fa[ fa[u][k-1] ][k-1]
// 含义：u 向上跳 2^k 步 = 先跳 2^(k-1) 步，再跳 2^(k-1) 步
for (int k = 1; k <= LOG; k++)
    for (int u = 1; u <= n; u++)
        fa[u][k] = fa[ fa[u][k-1] ][k-1];

查询过程

先将深度较大的节点上移到与另一节点相同深度
如果两节点已经相同，返回该节点
从大到小枚举 k，若 fa[u][k] != fa[v][k]，则同时上移 u 和 v
循环结束后，fa[u][0] 即为 LCA
  查询 LCA(7, 5)，假设树：
          1
         / \
        2   3
       / \   \
      4   5   6
     /
    7

  depth[7]=3, depth[5]=2, depth[4]=2

  Step 1: 将 u=7 上移到 depth=2
          从 k=1 开始: fa[7][1]=fa[4][0]=1 (depth[1]=0 < 2, 跳过)
          k=0: fa[7][0]=4, depth[4]=2 == 2, u=4
          现在 u=4, v=5, 同深度

  Step 2: 从 k=1 开始
          k=1: fa[4][1]=1, fa[5][1]=1, 相等, 不跳
          k=0: fa[4][0]=2, fa[5][0]=2, 相等, 不跳

  Step 3: 返回 fa[4][0] = 2, LCA(7,5) = 2

复杂度分析

预处理: O(n log n)
查询: O(log n)
空间: O(n log n)
LOG 的取值

LOG = ceil(log2(n))，通常取 __lg(n) + 1 或直接取 20（当 n ≤ 10^6 时足够）。

Tarjan 离线算法

核心思想

在一次 DFS 中，利用并查集 (DSU) 将已访问完的子树合并到祖先上。当处理查询 (u, v) 时，如果 v 已经被访问过，则 find(v) 就是它们的 LCA。

算法流程

将所有查询按节点分组存储（无向图，双向存）
从根节点开始 DFS
访问节点 u 时，标记为已访问，递归处理所有子节点
处理完子树后，将 u 合并到父亲：fa_union(u, parent[u])
对于每个与 u 相关的查询 (u, v)，若 v 已访问，LCA = find(v)
  树:       查询: LCA(4,5), LCA(7,8)
            1
           / \
          2   3
         / \   \
        4   5   6
       / \
      7   8

  DFS 过程:
  visit 1 -> visit 2 -> visit 4 -> visit 7
    finish 7, union(7,4)
    visit 8
      查询(7,8)? 7已访问, find(7)=4, LCA=4
    finish 8, union(8,4)
  finish 4, union(4,2)
  visit 5
    查询(4,5)? 4已访问, find(4)=2, LCA=2
  finish 5, union(5,2)
  finish 2, union(2,1)
  visit 3 -> visit 6
  finish 6, union(6,3)
  finish 3, union(3,1)

复杂度分析

总复杂度: O(n + q) * alpha(n)

其中 alpha 为反阿克曼函数，近似常数。比倍增法更快，但只能离线处理。

注意

Tarjan 离线算法需要一次性给出所有查询，无法在线回答。如果题目强制在线，应使用倍增法或树链剖分。

LCA 的应用

1. 树上两点距离

给定带权树，求节点 u 到 v 的距离：

int distance(int u, int v) {
    int l = lca(u, v);
    return depth[u] + depth[v] - 2 * depth[l];
}

对于带边权的树，可以用 dist[u] 表示根到 u 的距离：

int distance(int u, int v) {
    int l = lca(u, v);
    return dist[u] + dist[v] - 2 * dist[l];
}

2. 树上路径最大/最小边权

维护 maxEdge[u][k] 表示 u 向上跳 2^k 步路径上的最大边权：

// 预处理
maxEdge[u][0] = weight(u, parent[u]);
maxEdge[u][k] = max(maxEdge[u][k-1], maxEdge[ fa[u][k-1] ][k-1]);

// 查询路径最大边权
int pathMax(int u, int v) {
    int ans = 0;
    if (depth[u] < depth[v]) swap(u, v);
    for (int k = LOG; k >= 0; k--)
        if (depth[fa[u][k]] >= depth[v]) {
            ans = max(ans, maxEdge[u][k]);
            u = fa[u][k];
        }
    if (u == v) return ans;
    for (int k = LOG; k >= 0; k--)
        if (fa[u][k] != fa[v][k]) {
            ans = max(ans, max(maxEdge[u][k], maxEdge[v][k]));
            u = fa[u][k]; v = fa[v][k];
        }
    return max(ans, max(maxEdge[u][0], maxEdge[v][0]));
}

3. 树上差分

对 u 到 v 路径上的所有节点加 w：

int l = lca(u, v);
diff[u] += w;
diff[v] += w;
diff[l] -= w;
if (parent[l]) diff[parent[l]] -= w;  // 防止影响 l 的父节点

// 最后通过一次 DFS 自底向上累加

4. 两点是否在同一条链上

判断 u 是否在 a 到 b 的路径上：

bool onPath(int a, int b, int u) {
    return (lca(a, u) == u && lca(u, b) == u) ||
           (lca(b, u) == u && lca(u, a) == u);
}

C++ 完整模板

倍增法 LCA 模板

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 500005;
const int LOG = 20;

struct LCA {
    vector<int> adj[MAXN];
    int fa[MAXN][LOG + 1];
    int depth[MAXN];
    int n, root;

    void init(int _n, int _root) {
        n = _n; root = _root;
        dfs(root, 0);
        for (int k = 1; k <= LOG; k++)
            for (int u = 1; u <= n; u++)
                fa[u][k] = fa[ fa[u][k-1] ][k-1];
    }

    void dfs(int u, int p) {
        fa[u][0] = p;
        depth[u] = depth[p] + 1;
        for (int v : adj[u])
            if (v != p) dfs(v, u);
    }

    int query(int u, int v) {
        if (depth[u] < depth[v]) swap(u, v);
        // Step 1: 上移到同一深度
        for (int k = LOG; k >= 0; k--)
            if (depth[fa[u][k]] >= depth[v])
                u = fa[u][k];
        if (u == v) return u;
        // Step 2: 同时上移
        for (int k = LOG; k >= 0; k--)
            if (fa[u][k] != fa[v][k]) {
                u = fa[u][k];
                v = fa[v][k];
            }
        return fa[u][0];
    }

    int dist(int u, int v) {
        return depth[u] + depth[v] - 2 * depth[query(u, v)];
    }
};

Tarjan 离线 LCA 模板

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 500005;

struct TarjanLCA {
    vector<int> adj[MAXN];
    vector<pair<int,int>> qry[MAXN]; // {对方节点, 查询编号}
    int fa[MAXN], ans[MAXN];
    bool vis[MAXN];
    int n, root;

    int find(int x) {
        return fa[x] == x ? x : fa[x] = find(fa[x]);
    }

    void tarjan(int u, int p) {
        vis[u] = true;
        for (int v : adj[u])
            if (v != p) {
                tarjan(v, u);
                fa[v] = u;  // 合并到父亲
            }
        // 处理与 u 相关的所有查询
        for (auto& [v, id] : qry[u])
            if (vis[v]) ans[id] = find(v);
    }

    void solve(int _n, int _root, int q) {
        n = _n; root = _root;
        for (int i = 1; i <= n; i++) fa[i] = i;
        tarjan(root, 0);
        for (int i = 0; i < q; i++)
            printf("%d\n", ans[i]);
    }
};

例题与练习

模板题

P3379 【模板】最近公共祖先（LCA）
模板入门

LCA 基础模板题，给定一棵树和多组查询，求每组的 LCA。可用倍增法或 Tarjan 离线算法。

P1967 货车运输
模板中等

最大生成树 + LCA 求路径最小值。先建最大生成树，再用倍增法维护路径上的最小边权。

练习题

P3398 仓鼠找 sugar
中等

判断树上两条路径是否相交。关键结论：路径 (a,b) 和 (c,d) 相交当且仅当 LCA(a,b) 在路径 (c,d) 上或 LCA(c,d) 在路径 (a,b) 上。

P3128 [USACO15DEC] Max Flow P
中等

树上差分 + LCA 模板应用。对多条路径上的节点打标记，求最大标记次数。每次在 u→v 路径上 +1，用差分 + 子树累加实现。

P4281 [AHOI2008] 紧急集合 / 聚会
中等

三点 LCA 问题。三个点到某个汇合点的距离之和最小，汇合点为三个两两 LCA 中深度最大的那个。

P2680 运输计划
困难

二分答案 + 树上差分 + LCA。需要将一条边的权值改为 0，使所有路径的最大值最小。经典综合题。

P3258 [JLOI2014] 松鼠的新家
中等

树上差分 + LCA。按给定顺序访问节点，对相邻节点间路径上的所有节点 +1，最后还原统计。

P5836 [USACO19DEC] Milk Visits G
中等

树上路径查询，判断路径上是否存在指定颜色的节点。可用 LCA + 并查集或树链剖分。

变种题

P1600 [NOIP2016 提高组] 天天爱跑步
困难

树上差分 + LCA 综合难题。每条路径上的观察员在特定深度能看到选手，需要分别讨论上行和下行路径。

CF 208E Blood Cousins
中等

给定多组查询 (v, p)，求 v 的第 p 个祖先的所有后代中深度与 v 相同的节点数。离线 + 启发式合并或 DFS 序 + 二分。

CF 609E Minimum spanning tree for each edge
中等

对每条边求包含该边的最小生成树权值。结合 Kruskal 和倍增 LCA 求路径最大边权。

CF 500E New Year Domino
困难

区间覆盖 + 贪心 + 倍增。将多米诺骨牌倒塌问题转化为区间覆盖，用倍增跳过已覆盖区间求代价。

P4103 [HEOI2014] 大工程
困难

虚树 + LCA。在树上选定 k 个关键点建虚树，求关键点两两之间的距离和、最大值、最小值。

扩展题

P2420 让我们异或吧
入门

树上路径异或和。预处理根到每个节点的异或和，利用 LCA 性质 O(1) 求路径异或和。

P3320 [SDOI2015] 寻宝游戏
困难

动态维护虚树周长。每次插入/删除关键点，用 set 维护 DFS 序相邻点对的 LCA 距离和。

CF 293E Close Vertices
困难

点分治 + 树状数组。统计树上距离不超过 l 且边权和不超过 w 的点对数量。

中等

点分治模板题。统计树上距离恰好为 k 的点对是否存在。点分治的核心框架练习。

困难
